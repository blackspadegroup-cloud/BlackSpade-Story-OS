## ACT-01｜命运开始交错

### Theme
命运开始交错，但没有任何人意识到。

### Episode Blueprint

| # | 标题 | 一句话概要 |
|---|------|-----------|
| EP01 | 一杯香槟，改写三个人的命运 | 意大利卡普里岛的超级游艇生日派对上，林知夏用一杯香槟解决危机，也第一次进入陆景深的视线。 |
| EP02 | 棋手的第一步 | 陆景深没有责怪她，反而提出一场合作，让林知夏第一次觉得这个男人完全无法预测。 |
| EP03 | （待定） | 陆景深正式邀请林知夏加入陆氏未来项目，但她认为自己不需要任何人的施舍。 |
| EP04 | （待定） | 林知夏拒绝合作，坚持靠自己的能力证明价值；陆景深没有挽留，只默默开始布局。 |
| EP05 | （待定） | 生日派对结束后，顾言川与林知夏第一次真正认识，两人自然轻松的相处，与陆景深形成鲜明对比。 |
| EP06 | （待定） | 林知夏结束意大利工作返回吉隆坡，一通电话打破平静，她以为自己闯祸了。 |
| EP07 | （待定） | 她没有被开除，而是被调往陆氏集团负责新项目，却不知道这一切都是陆景深的安排。 |
| EP08 | （待定） | 林知夏第一天进入陆氏，凭实力完成会议，却因此成为办公室众人针对的对象。 |
| EP09 | （待定） | 面对林知夏被针对，陆景深选择不出手，因为他知道，现在帮助她，只会害了她。 |
| EP10 | （待定） | 午休时，顾言川再次偶遇林知夏；与此同时，陆景深第一次在远处看见顾言川，却不知道他是谁。 |
| EP11 | （待定） | 匿名的咖啡与光明正大的咖啡前后出现，陆景深第一次意识到，棋盘上出现了新的变量。 |
| EP12 | （待定） | 林知夏开始怀疑自己进入陆氏并非巧合，决定主动寻找真相，而陆景深知道，她终于开始下这盘棋。 |
| EP13 | （待定） | 陆景深第一次单独与林知夏讨论项目，没有一句多余的话，却让她第一次意识到，两人的思维方式惊人地相似。 |
| EP14 | （待定） | 顾言川邀请林知夏拍摄纪录片素材，让她暂时放下工作，也第一次重新感受生活本身。 |
| EP15 | （待定） | 一场陆氏商业酒会，陆景深、林知夏、顾言川第一次出现在同一个空间；三个人互相影响，却没有任何人意识到，一场改变人生的棋局已经正式开始。 |

### Act Goal
建立三位主角，建立办公室主线，建立双男主独立感情线，双男主彼此不知道对方存在，林知夏不知道自己正在改变两个人的人生。

### Act Ending
三位主角首次出现在同一事件中，但两位男主仍未正式认识彼此。
