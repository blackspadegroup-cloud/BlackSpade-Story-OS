## Module 13 — Character Visual Language（角色视觉语言）

### Visual Symbol（视觉符号）
围棋残局

### Signature Action（标志性动作）
每做一个重要决定，落下一颗棋子

### Signature Object（标志性物件）
办公室里的围棋残局（整部剧不会下完）

### Visual Motif（视觉主题）
围棋不是兴趣，是思考方式。每落一子代表一个布局完成。从不解释。直到最终章观众才会理解整盘棋的意义。

### Production Rules（制作规则）

| 规则 | 说明 |
|------|------|
| 触发条件 | 每次重大布局完成落一子 |
| 表现方式 | 不刻意强调，不解释原因 |
| 镜头语言 | 镜头停留 1-2 秒即可 |
| 叙事作用 | 观众到最终章才能理解整盘棋的意义 |
