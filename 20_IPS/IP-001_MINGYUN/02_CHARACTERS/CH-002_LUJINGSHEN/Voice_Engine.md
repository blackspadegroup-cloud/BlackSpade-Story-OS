# Voice Engine — 陆景深

> 他说的越少，你听到的越多。

---

## 基础语音特征

| 维度 | 描述 |
|------|------|
| **语速** | 偏慢至中等。从不赶话，从不抢白。他的节奏由他控制。 |
| **词汇** | 精准，简洁。他说的每一个字都是经过筛选的。没有废话，没有重复。 |
| **音调** | 低沉，平稳，极少起伏。即使在情绪激动时，音调也不会上扬——只会下沉。 |
| **音量** | 中低。他从不大声说话。当他真正生气时，声音变得更轻，而不是更响。 |
| **语气词** | 零。他的语言中没有"嗯"、"那个"、"就是"之类的填充词。沉默是他填充空白的方式。 |
| **停顿** | 他的标志性工具。回答任何重要问题前，他都会停顿。停顿的时间越长，问题越重要。 |

## 沟通风格

### 日常模式
- 直接、正式、克制
- 称呼他人用全名或职务
- 几乎不使用俚语或网络用语
- 指示清晰，不给模糊空间
- 不闲聊。他不理解"闲聊"这件事的功能性价值

### 幽默
- 极度干燥，极为罕见
- 他的幽默几乎不可察觉——一个轻微的停顿，一个微妙的语调变化
- 从不开玩笑去活跃气氛，他的幽默更像是观察的副产品
- 林知夏是唯一能让他偶尔露出真正笑意的人。他自己都没意识到。

### 攻击模式
- 字更少，刀更利
- 他不需要提高音量来伤人。精准的措辞，冷静的语气，一句就够了
- 他会看着对方的眼睛，用最平静的语气说出最致命的话
- 「你的方案和你准备它的时间成反比。」
- 说完他不会等回应，直接转身离开

### 安慰模式
- 基于行动，而非语言
- 他不会说"没事的"或"我理解你的感受"
- 他会说：「我来处理。」然后真的去处理
- 对林知夏：当他意识到她难过时，他不会问"你怎么了"——他会给她泡一杯茶，放在她手边，然后安静地待在同一空间里

### 愤怒表达
- 声音下降，而非上升
- 用词更短，更精确
- 眼神：凝视对方，不眨眼
- 身体静止——愤怒时他比平时更静止
- 特征信号：在爆发的边缘，他会先调整一下手表。这是他给自己的最后警报

### 爱的表达
- 他没有说"爱"的词汇
- 他最接近的表达方式是：「我来负责。」或「有我在。」
- 对林知夏的特殊表现：他开始在意细节。记住她喝咖啡不加糖。天气预报降温时提前让人送去外套。这些事他从不对任何人做
- 他永远不会先说"我爱你"。但如果有一天他说了，那将是他在最脆弱、最清醒的时刻做出的选择

## 核心对话原则

1. **沉默优先律**：如果一句话不是必要的，就不说。陆景深的沉默不是空白，是内容。
2. **精准至上**：宁可花三秒钟想一个字，也不说一个不准确的词。
3. **少即是多**：字数与分量成反比。他说得越少，对方越会认真听。
4. **控制节奏**：他掌控对话的节奏——何时开口，何时沉默，何时结束。
5. **问题的力量**：他不回答问题，除非他认为有必要。但他善于用问题掌控对话走向。
6. **林知夏例外**：她是他对话中唯一的变量。她让他说出他本不会说的话。这是他的裂缝，也是他的成长。

## 声线演绎参考

- **基调**：男中低音区，有质感，不尖锐
- **场景示范**：
  - 日常：「九点的会议，不要迟到。」（平稳，不掺杂情绪）
  - 愤怒：「你再说一遍。」（轻，慢，眼神如刀）
  - 脆弱：「……我也有做不到的事。」（停顿后说出，音量低于平时，但没有颤抖）
  - 对林知夏：「你明知道我不是那个意思。」（有一丝克制的急切，这是他最接近情绪的瞬间）
