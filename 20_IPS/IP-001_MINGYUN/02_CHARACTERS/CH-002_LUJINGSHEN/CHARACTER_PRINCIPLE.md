# CH-002 陆景深 — Character Principle

> 这是 CH-002 陆景深的最高角色原则。
> 以下所有 Layer 均不得违反此原则。

---

**他穷尽一生追求掌控，却在爱情面前第一次承认：有些东西，越想控制，越会失去。**

**核心命题：** 控制（Control）

---

## 原则解析

| 维度 | 含义 |
|------|------|
| **核心驱动** | 不是权力欲，不是控制狂的表象。是恐惧——对失控的深层恐惧。 |
| **控制的意义** | 对他来说，控制 = 秩序 = 安全 = 可预测。失控 = 混乱 = 脆弱 = 受伤。 |
| **外在表现** | 商业上运筹帷幄，情绪上滴水不漏，生活上精密规划。所有人都说他冷静、果断、无懈可击。 |
| **内在真相** | 他的控制不是强大——是防御。他害怕的不是失败，而是失控后的自己。 |
| **脆弱面** | 林知夏是他人生中第一个"不受控变量"。她不在乎他的地位，不畏惧他的气场，不需要他的资源。他没办法用任何他熟悉的方式去"处理"这段关系。 |
| **成长方向** | 从"控制一切以求得安全感"到"承认有些最珍贵的东西，恰恰在掌控之外"。 |

---

## Signature Object

**手表。**

意义：陆景深透过时间建立秩序。透过秩序获得控制感。

每当面对压力、风险或情绪波动时，他都会下意识看向手表或调整手表。

这是未来所有作品必须保持一致的重要视觉符号。

---

## Core Arc

```
控制世界 → 控制别人 → 控制自己 → 
遇见林知夏（第一次失控）→ 学会信任 → 
尊重选择 → 放下控制 → 真正成熟
```
