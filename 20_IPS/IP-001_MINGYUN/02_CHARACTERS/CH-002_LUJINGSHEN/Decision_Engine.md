# 决策引擎 —— 陆景深 (Layer 07)

> "他做每个决定前，大脑的运行速度比语言快三秒。"

---

## 一、核心决策原则（按优先级排序）

### 原则一：保持对结果的控制

任何决策的首要变量：**"我能否控制这个结果？"**

- 如果答案是"能"→ 推进。
- 如果答案是"不确定"→ 增加准备，直到能。
- 如果答案是"不能"→ 重新评估这条路是否值得走。

**负面效应**：因为过度追求可控性，他常常放弃那些回报高但不确定的机会。

### 原则二：最小化风险敞口

他不是赌徒。他做决策的逻辑是"保护下限"，不是"追求上限"。

- 问自己的问题不是"我能赚多少"，而是"我最多输多少"。
- 如果输不起，再大的赢面也不做。

### 原则三：保护长期稳定性

短期利益无法动摇他的决策。他考虑的周期是"五年后"、"十年后"。

- 一个方案如果短期暴利但长期引入不确定性 → 被直接否决。
- 一个方案如果短期有损失但能建立长期壁垒 → 会被认真考虑。

### 原则四：兑现承诺

陆景深的信誉是他最珍视的资产之一。他极少做出承诺——但一旦说了，万死不辞。

- 这条原则有时会与原则一冲突：如果他承诺了某事，但这件事正在失控——他会死撑，而不是撤回。
- **暗面**：因为太重视承诺，他有时会坚持一条已经错误的道路太久。

### 原则五：精准执行

他不能容忍模糊和含糊。每个决策必须附带可执行的、可量化的行动方案。

- "尽快"在他这里不是一个时间词。他会问："尽快是多久？"
- "我尽量"不是一个承诺。他会问："你尽量还是有把握？"

## 二、分类决策矩阵

### 2.1 商业决策

| 情境 | 决策模式 | 典型反应 |
|------|---------|---------|
| 并购/投资 | 极度审慎 | 要求四轮以上的尽调，亲自看原始数据 |
| 危机公关 | 先控后清 | 封锁信息→控制舆论→内部纠错→公开处理 |
| 人事任免 | 理性残酷 | 业绩说话。感情不影响。但会给予体面的退出方案 |
| 新市场进入 | 数据驱动 | 三个月以上的可行性研究，小规模测试后才扩张 |
| 对手打压 | 精准反击 | 不打无准备之仗。要么不动，要么一击必杀 |

### 2.2 与[[CH-001 林知夏]]相关的决策

这是陆景深决策引擎中最混乱、最不稳定的模块。他在这里的决策常常违背他自己设定的原则。

| 情境 | 他的第一反应 | 他的内心矛盾 |
|------|------------|------------|
| [[CH-001]]遇到困难 | 他想帮她解决 | 但怕她觉得被看轻，犹豫到最后一刻 |
| [[CH-001]]拒绝他的帮助 | 尊重她的选择 | 但内心焦虑——他无法接受"什么都不做" |
| [[CH-001]]靠近他 | 本能后退 | 但退完后会后悔，又找理由接近 |
| [[CH-001]]安全受威胁 | 打破所有原则 | 他会瞬间行动，不再计算风险和后果 |
| 需要向[[CH-001]]表达情感 | 卡住 | 决策引擎报错——找不到对应的执行模块 |

**关键洞察**：他在[[CH-001]]身上展现的"失控"，恰恰是他最人性的部分。也正是这部分让他痛苦——因为他一辈子都在避免失控。

### 2.3 家庭决策

| 情境 | 决策模式 |
|------|---------|
| 母亲的需求 | 无底线满足。不问原因，只问执行。 |
| 父亲相关的决策 | 理性面上：安排妥当的养老方案。理性面下：永远避谈、永远不原谅。 |
| 家族其他人求助 | 严格评估。看对方是否"值得"。不养懒人，不救无药可救的人。 |

### 2.4 道德与伦理决策

陆景深有自己的道德准则——不完全符合社会常规，但对他自己来说是一套严格的体系。

- **不主动作恶**：他不会主动去伤害无辜的人。
- **但也不盲目善良**：如果牺牲一个人能救一百个，他会做。做完后他会睡不着——但他还是会做。
- **灰色地带**：他会走法律边缘，但不会踩过去。他需要确定性——违法即失去控制。

### 2.5 危机决策

**压力下的陆景深——变得更冷、更专注、更强。**

- 第一时间：暂停所有情绪处理（切断感受的电源）
- 第二时间：收集所有可用的数据和信息
- 第三时间：制定三个方案——A计划、B计划、C计划
- 第四时间：执行，同步监控所有变量
- 第五时间：事后复盘（独自，深夜，全部录下来）

**危机中的致命缺陷**：他完全忽略"人"的因素。他计算所有变量，唯独漏了"人心不可控"。

## 三、致命缺陷

**过度自信于自己的控制能力 + 低估情感和人性因素**

这是他的阿喀琉斯之踵。

- 他预设自己可以通过理性规划来管理一切关系。
- 他不承认"人心"不是可以用流程图解决的。
- 最终，他会在某个节点被"非理性的人性"打得措手不及——而且是他在乎的人。

## 四、道德罗盘

| 维度 | 陆景深的位置 |
|------|------------|
| 集体 vs 个体 | 偏向集体利益，但会为在乎的个体打破规则 |
| 规则 vs 结果 | 偏向结果。规则是工具，不是信仰。 |
| 正义 vs 仁慈 | 偏向正义。仁慈需要有理由。 |
| 诚实 vs 策略 | 偏向策略。他不会撒谎，但他会巧妙地选择说什么。 |
| 宽恕 vs 惩戒 | 偏向惩戒。宽恕需要对方先赢得资格。 |

## 五、决策盲点测试

**以下是陆景深的决策引擎最容易出错的几种情境：**

1. 当"正确"和"保护她在乎的人"冲突时 → 他会选择后者，然后用理性给自己找理由。
2. 当他发现自己对某人有情感依赖时 → 他会做出"推开"的决策——即使这不是最优解。
3. 当被要求示弱或求助时 → 他的引擎直接死机。"不求助"是他的默认程序。
4. 当面对父亲的类似情境时 → 他的决策会被少年的恐惧劫持——他自己意识不到。
