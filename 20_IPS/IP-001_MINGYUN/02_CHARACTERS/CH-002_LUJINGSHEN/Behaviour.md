# 行为图谱 —— 陆景深 (Layer 08)

> "他的每一个动作都在说：我在场。我控制。我不需要更多。"

---

## 一、默认模式

陆景深的默认状态是一个不动的人。

- **姿态**：坐姿笔直但不僵硬。双手通常安静地放在桌面或交叠在身前。极少靠在椅背上。
- **眼神**：专注。不眨眼的时间比常人略长。注视时有一种"在把你装进档案"的分量感。
- **声音**：音量中等偏低，语速均匀，几乎没有语气起伏。生气时音量**不升反降**。
- **静止能力**：他可以在一个位置坐四十分钟不动，不需要调整姿势、玩手机或做任何小动作。这种静止不是僵硬——而是**高度的在场感**。

---

## 二、习惯性行为

### 2.1 标志性动作：调整腕表

这是他最显著、最频繁的微动作。

**触发条件**：
- 感到压力或焦虑时
- 面临需要立刻做决定的时刻
- 在交谈中需要打断对方节奏时
- 听到不想听的对话时
- 在[[CH-001]]面前不知如何回应时

**动作细节**：左手抬起，拇指和食指捏住表冠，轻轻旋转调整——有时根本没对准时间，只是**转动**那个动作本身。那一刻他在争取0.5秒的思考时间。

**深层含义**：那块表是父亲在他14岁生日送的。调整它是无意识的，但每一次都是在与那个少年对话。

### 2.2 时间行为

**到场永远准时**。
- 不早到（早到是浪费时间）
- 不迟到（迟到是失控的表现）
- 他会精确计算交通时间，把一切变量（堵车、电梯等待、安保检查）都纳入公式
- 如果因不可抗力迟到了——他不会道歉，但会用行动补回那些时间（压缩会议，但效率加倍）

**他对别人迟到的容忍度：零容忍**。
- 第一次：一个眼神，不说话。
- 第二次：安排下次会议时，给对方的预留时间缩短50%。
- 第三次：这个人不会再出现在他的日程上。

### 2.3 空间与物件的秩序

- 办公桌永远整洁。文件归档、笔具排列、水杯放在固定位置。
- 会议前会花30秒整理面前的物件——键盘摆正、笔记本对齐、笔放回笔槽。
- 家里：极简主义风格。没有杂物。每样东西都有固定位置。
- 他不能忍受"无序"——对他来说，外部秩序是内心秩序的延伸。

### 2.4 阅读与信息处理

- 习惯站着读报告。当他站着读东西时，说明他在**认真对待**这份材料。
- 阅读速度极快，但会反复读关键段落后半页才翻。
- 读到重要信息时不会做标记——他会**合上文件，闭眼三秒**，把信息存入脑中，再继续。

## 三、身体语言详解

### 3.1 常态肢体语言

| 部位 | 表现 | 含义 |
|------|------|------|
| 手 | 极少做手势。安静放在桌面或交叠 | 控制表达欲，不浪费动作 |
| 肩膀 | 始终挺直，不倚靠 | 高度的自我要求和警戒状态 |
| 腿/脚 | 双腿自然分开，极少交叉 | 稳定、准备随时行动 |
| 头部 | 微仰，不低头 | 自信、审视、不妥协 |
| 眼神 | 注视时间长，不躲闪 | 既是一种尊重，也是一种测量 |
| 嘴 | 嘴角微微平直，很少笑 | 他不是不高兴——他只是不觉得需要笑 |

### 3.2 倾听状态

当他认真听你说话时，他会：
- 完全静止。从头到脚都不动。
- 眼神锁在你身上，几乎不眨眼。
- 不插话。等你把话全部说完——甚至等你说完后的沉默结束——才会开口。

**这是一种极具压迫感的倾听方式**。很多人会在他面前不自觉地说更多、交代更多，因为他什么都没做，你却感觉自己被看透了。

### 3.3 身体接触

**极其罕见**。他几乎不主动触碰任何人。

- 握手：干脆、有力、一秒内松开。
- 被触碰：如果有人在对话中碰他的手臂或肩膀——他的第一反应是微不可察的僵硬。他不会躲开，但你明显感到他"离开了"一下。
- 对[[CH-001]]的特殊例外：他会在极少数时刻——比如她没注意的时候——看她很久。但他仍然不会主动碰她。他的克制在这一点上近乎固执。

## 四、压力反应

### 4.1 压力分级表现

| 压力等级 | 外部表现 | 内部状态 |
|---------|---------|---------|
| 低（日常） | 一切正常，准时，有序 | 稳定运行 |
| 中（项目危机） | 更安静、更冷、减少社交 | 高速运转，情绪处理被挂起 |
| 高（重大失控） | 完全沉默、消失、不接电话 | 进入"作战模式"——切断所有感受 |
| 极高压（触及创伤） | 调整腕表频率暴增、深夜独处 | 少年恐惧被激活，但表面更冷 |

**关键模式**：压力越大，他表现出的控制越多，而不是越少。

### 4.2 情绪行为对应表

| 情绪 | 外在行为 |
|------|---------|
| 愤怒 | 语调变冷，用词更精准，眼神更直接。不会提高音量，但每句话都像刀。 |
| 焦虑 | 整理桌面、调整腕表、检查数据、反复确认行程。同时会喝更多的黑咖啡。 |
| 悲伤 | 隔离。取消所有社交/非必要安排。独自在书房，不开灯。拒绝任何人的关心。 |
| 幸福感 | 难以捕捉。唯一信号：肩膀的紧张感略略释放，呼吸变深一点点。"满意"是他表达幸福的方式。 |
| 嫉妒 | 他不会承认。也不会表现出来。但他会给[[CH-001]]身边的人分配更多的工作。 |
| 困惑 | 轻微的眉心纹路出现（极短暂），然后恢复正常。他会说："我需要更多信息。" |

## 五、偏好清单

### 5.1 喜欢

| 类别 | 具体偏好 | 原因剖析 |
|------|---------|---------|
| 秩序 | 一切都井井有条 | 混乱触发他的不安 |
| 安静 | 没有背景噪音的空间 | 安静让他能思考 |
| 精准 | 时间、数据、语言都要精确 | 模糊意味着风险 |
| 古典音乐 | 巴赫、勃拉姆斯、肖邦 | 有结构、有逻辑、有秩序 |
| 单一麦芽威士忌 | 艾雷岛产区，泥煤味 | 足够强烈，不需要复杂的调酒来掩盖 |
| 海洋 | 站在岸边看海，不说话 | 海是无尽但可控的——他知道潮汐规律 |
| 黑咖啡 | 不加糖不加奶 | 纯粹的、不妥协的味道 |

### 5.2 不喜欢

| 类别 | 具体厌恶 | 为什么 |
|------|---------|-------|
| 惊喜/意外 | 任何没有事先通知的事 | 等于失控 |
| 迟到 | 不守时的人 | 等于不尊重规则 |
| 低效 | 开会没议程、说话没重点 | 浪费他的时间 |
| 情绪爆发 | 大声争吵、哭闹、失控 | 让他不知所措，但不会表现出来 |
| 被触碰 | 意外的身体接触 | 侵入他的安全边界 |
| 计划被打乱 | 临时改期、取消、变动 | 需要重新调整整个系统 |

## 六、标志性场景快照

### 场景一：会议前

他不与人寒暄。他走进会议室，放下公文包，拿出笔记本，摆正笔。花10秒扫视在场的所有人——不是为了打招呼，而是为了**确认在场的人头与名单一致**。然后坐下，抬手调整腕表，说："开始。"

### 场景二：深夜办公室

经理层早已下班。只有他的办公室灯还亮着。他没有在加班——他坐在椅子上，看向窗外，手边是一杯已经凉了的黑咖啡。他没有在想任何事情。他在"停"——这是他唯一允许自己停止控制的时间。

### 场景三：在她面前

[[CH-001]] 说了一些让他措手不及的话。他停顿了1.5秒——比正常反应多了一秒。在那1.5秒里，他调整了腕表、垂下目光（罕见地移开视线）、然后再抬起眼时，表情已经恢复如常。他说："嗯。" 然后转移了话题。
