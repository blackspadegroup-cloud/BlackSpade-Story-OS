# CH-002 陆景深 — 角色快照

> **一句话介绍：** 他是那个永远站在权力中心的人——直到遇见一个他看不懂、握不住、又放不下的女人。他学会的，不是如何掌控她，而是如何放手。

**原则：** 他穷尽一生追求掌控，却在爱情面前第一次承认：有些东西，越想控制，越会失去。
**核心主题：** 控制（Control）
**DNA：** 掌控 · 理性 · 克制 · 精准 · 孤独

---

## 一、核心驱动力（Core Drive）

| 维度 | 内容 |
|------|------|
| **Want（渴望）** | 完全的掌控——对局势、对人生、对身边的一切。想要她属于他。 |
| **Need（需要）** | 学会放手——尊重她的独立性，接受有些事不是靠计划和执行力就能解决的。 |
| **Wound（创伤）** | 十六岁家道中落，目睹父亲从顶峰坠落成一蹶不振的失败者。他发誓绝不让同样的事发生在自己身上。 |
| **Lie（谎言）** | "只要我控制一切，我就安全了。" |
| **Truth（真相）** | 真正的力量不在于掌控他人，在于掌控自己——而最大的失控往往最值得经历。 |

---

## 二、当前弧线（Current Arc）

### Season 1：初遇 — 不可控变量

**起点：** 陆景深正在运作职业生涯中最大的一笔收购。一切按计划推进——然后他遇到了对方公司的代表律师，林知夏。

**弧线描述：**
- 第一阶段：她只是"需要处理的一个变量"——他以为可以像以往一样，用逻辑、资源和耐性解决问题
- 第二阶段：他发现她不吃这一套。她不怕他，不讨好他，不需要他
- 第三阶段：他开始不自觉地调整行程、改变习惯、做他自己都解释不了的事——只是为了「遇到」她
- 弧线高潮：他说出了那句连自己都没想到会说的话——"……你明天还来吗？"

**当前状态：** S1 中段。拉锯期。他正在从"想掌控她"慢慢过渡到"想了解她"——但他自己还没有完全意识到这个转变。
