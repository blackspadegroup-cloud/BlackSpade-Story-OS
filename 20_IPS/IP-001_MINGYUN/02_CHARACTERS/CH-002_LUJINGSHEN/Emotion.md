# 情绪图谱 —— 陆景深 (Layer 05)

## 一、情绪基线

**常态情绪基调：冷静、克制、测量感**

陆景深的情绪不是"没有"，而是"不释放"。他所有的情绪都被一个精密的内在系统过滤、压缩、归档。外人看到的是一张波澜不惊的脸，但那张脸下面是一个始终在计算、在预警、在排序的引擎。

> "他很少大声说话。他不需要。安静本身已经足够有力量。"

---

## 二、情感模块分项

### 2.1 爱情图谱

从无恋爱史。不是因为没有机会——恰恰相反——而是因为他**不信任自己无法控制的东西**。

- **对关系的认知**：将恋爱视为一组无法完全量化的变量。变量太多，失控概率太高，因此主动回避。
- **过往经历**：有过短暂的交往关系，均以他"情感无法投入"告终。对方的共同反馈："他很完美，但感觉不到他在那里。"
- **对[[CH-001 林知夏]]的特殊性**：她是第一个让他**想要靠近、却无法掌控**的人。这种矛盾让他既吸引又恐惧。他无法用惯常的手段——金钱、资源、控制——来赢得她，这暴露了他情感工具箱的匮乏。
- **心动信号（极其微弱，但存在）**：在[[CH-001]]转身离去时，他会多看她背影一秒钟。在她不需要他帮助时，他会无意识地转动腕表。

### 2.2 友谊图谱

朋友极少。且"朋友"在他字典里的定义极其苛刻：通过了至少五年以上的信任累积。

- **社交风格**：保持职业距离。不参加非必要的社交。不透露个人信息。
- **为何难以交到朋友**：他擅长看穿别人的动机，这让他对大部分人心存戒备。他极少主动示弱，而不示弱的关系永远停留在表面。
- **信任门槛**：几乎不可能完全获得。需要一个在重大危机中证明自己不离开、不背叛的人。目前他的圈子里——没有任何人达到这个标准。

### 2.3 家庭图谱

**关于母亲**：敬重，但距离。母亲在他父亲出事后的坚韧让他敬佩，但她的沉默也让他无法真正靠近。他会定期转账、安排一切、确保她生活无忧——这是他说"我爱你"的方式。

**关于父亲**：这是陆景深心中唯一没有愈合的伤口。

- 父亲曾是成功的商人，企业倒闭后一蹶不振。
- 少年陆景深亲眼看着那个曾经高大的男人一点点崩塌。
- **核心情绪**：愤怒——不是恨父亲失败，而是恨他**输后认命**。恨他没有战斗到底，恨他让母亲哭。
- 成年后，陆景深做的每一个决定，几乎都在潜意识里告诉自己一句话：**"我不能像他一样。"**

### 2.4 信任与安全感

- **信任难度**：极难。信任对他是一种"交出控制权"的行为，等同于把自己放在了一个可能受伤的位置。
- **唯一的安全感来源**：**控制**。不是权力，不是金钱——而是对局势、对结果、对他人的行为的**预判和掌控**。这是他内心唯一的锚。
- **不被理解的痛苦**：外人看他是一个控制狂霸总，但控制对他而言不是欲望，是**生存需要**。

## 三、情绪表达模式

### 3.1 表情与语言

**几乎为零的外显情绪表达**。他的脸是一面墙。

| 内心状态 | 外在表现 |
|---------|---------|
| 愤怒 | 语气更冷、用词更精准、音调不变 |
| 焦虑 | 开始整理桌面、调整手表、检查数据 |
| 悲伤 | 消失——他会推掉所有非必要的安排，独自待着 |
| 快乐 | 几乎无法从他脸上读到。唯一的信号：肩膀微微放松零点几秒 |
| 震惊 | 最长0.5秒的停顿，然后恢复正常 |

### 3.2 他表达关心的方式

他几乎从不说"你还好吗"或"我担心你"。他说的是：

- "这件事我来处理。"
- "以后不要一个人去这个地方。"
- "你需要什么资源，告诉我。"
- "我查过了，那家医院的胸外科排名前三。"

**他的爱藏在行动里**。他为你移除障碍，让你的一切顺利运转。这就是他情感词汇的全部。

### 3.3 情绪触发点

| 触发事件 | 反应 |
|---------|------|
| 被操纵/被欺骗 | 立刻进入战斗模式——冷静、致命、不留余地 |
| 被看见弱点 | 防御性沉默、转移话题、物理远离 |
| 被强迫暴露情感 | 情绪彻底关闭，变成一台机器 |
| 失去一个精心准备的局势 | 表面冷静，内里剧烈反应——会复盘到深夜 |
| [[CH-001]]在他面前不依赖他 | 困惑 + 隐约的失落（他自己都不承认） |

## 四、情感发育诊断

陆景深的情感发育在**少年时代父亲的失败后**就停滞了。他在情感层面一直没有真正成长——他学会了控制一切，除了自己的内心。

- **缺失的能力**：示弱、接受关心、表达脆弱、原谅失败（包括原谅他自己）。
- **需要突破的关卡**：承认"我害怕"比说"我能搞定"更需要勇气。
- **情感成长的终点**：当他能够对[[CH-001]]说出一句"我需要你"的时候。
