# Performance — 陆景深

> 他不是没有情绪。他只是把情绪管理到了别人看不见的地方。

---

## 能量水平表

| 状态 | 能量等级 | 外在表现 | 场景示例 |
|------|---------|----------|----------|
| **完全控制** | ⚡⚡⚡⚡⚡ | 气场全开，决策果断，眼神锐利。每一个动作都精确。 | 董事会、谈判桌、危机处理 |
| **日常运作** | ⚡⚡⚡⚡ | 标准CEO模式。高效、得体、稍有距离感。 | 日常工作、会议、商务午餐 |
| **轻度疲劳** | ⚡⚡⚡ | 话更少，停顿更长。不参与非必要社交。 | 连续工作12小时后、长途出差归来 |
| **消耗警戒** | ⚡⚡ | 几乎不说话。只做必要回应。脸上的表情更少。开始调整手表。 | 重大决策后、面对[[CH-001]]的情感场景 |
| **真正脆弱** | ⚡ | 唯一可能出现在深夜或私人空间。他没有能力维持完美面孔。 | 面对父亲相关记忆时、与[[CH-001]]的深度对话后 |

## 表演元素

### 凝视
- **基线**：坚定、穿透性强。他注视人时不眨眼的时间长于常人
- **含意**：在对话中持续凝视=「我在分析你」。在工作中短暂移开视线=「我已经决定了」
- **脆弱信号**：他只有在无法面对的时刻才会看向别处——通常是看向自己的手表
- **对[[CH-001]]的特殊变化**：她的视线会让他在某些时刻下意识移开目光。他自己都没意识到

### 呼吸
- **基线**：控制的、均匀的。呼吸频率从不因外部压力而变化
- **应激信号**：唯一的变化是——在开口回应重要问题之前，有一次轻微的、更深的吸气
- **核心含义**：他的呼吸控制是他"外表永远平静"的物理基础。如果有一天他的呼吸乱了，说明他的世界也乱了

### 停顿
- **他最强大的表演工具**
- **种类**：
  | 停顿类型 | 时长 | 含义 |
  |----------|------|------|
  | 思考停顿 | 1-2秒 | 分析信息、组织措辞 |
  | 权力停顿 | 2-4秒 | 让对方在他的沉默中感受到压力 |
  | 情感停顿 | 3-5秒 | 他在压制某种情绪。停顿越久，情绪越强 |
  | 抉择停顿 | 5秒+ | 他在做一个艰难的决定。这是观众值得注意的时刻 |
- **叙事功能**：他的停顿是观众理解他内心的重要窗口。他越沉默，意味着内部越激烈

### 微表情
- **总体特征**：他几乎不做多余的面部表情。他的脸是一道紧闭的门
- **可观测的微小变化**：
  - **愤怒**：下颌轻微收紧（极难察觉）
  - **不耐烦**：鼻翼轻微扩张（需要特写镜头才能捕捉）
  - **温暖**：眼角周围的肌肉几乎不可察觉地松弛——这是他最接近"笑"的版本
  - **痛苦**：完全静止。他痛苦时脸上没有任何表情。那是最大的表演
  - **对[[CH-001]]的例外**：在她面前，这些微表情出现的频率更高、持续的时间更长

### 身体语言
- **总体姿态**：挺拔但不僵硬。站姿稳定，重心居中
- **手势**：极少。他说话时手几乎不动。当他做手势时，动作小而有控制
- **空间**：他自然占据比他身体需要更多的空间——不是侵略性的，而是习惯性的"这里由我决定"
- **进入房间**：他的进入方式是一致的——步伐不变，目光扫视全场，不需要说一句话就改变了房间的能量场
- **离开**：他离开时不需要告别。他的离开和他的存在一样安静

### 声音表演
- **情感场景的声线法则**：
  - 情绪越强烈，音量越轻——不是虚弱，是更深的控制
  - 重要台词前的停顿——是他给自己信号："准备好，接下来这句话会让你暴露"
  - 他在讲真话和讲策略性的话时声音没有任何区别——这是他的天赋，也是他的悲哀
  - 唯一的声音裂缝：面对[[CH-001]]时，偶尔会有一个词的结尾出现极其轻微的颤抖。可能只有一个音节，但那是他真正在意的证明

## 标志性动作：调整手表

**这是陆景深的标志性动作——他所有真实情绪的出口。**

| 场景类型 | 动作细节 | 内部含义 |
|----------|----------|----------|
| 压力下 | 下意识转动表盘 | 「我需要控制住局面。」 |
| 情感暴露前 | 拇指轻擦表镜边缘 | 「接下来要说的话可能会暴露我。」 |
| 面对[[CH-001]] | 调整完手表后，把手放下。这个动作变慢了 | 「我在她面前不自觉地卸下防备。」 |
| 重大决定 | 解开表扣——再扣上 | 「这个选择在动摇我的根基。」 |

**叙事功能**：对观众来说，手表动作是他平静外表下的唯一预警系统。当陆景深摸手表时，意味着一件事——在面具之下，有什么正在发生。

## 表演核心原则

1. **控制本身就是表演**：他的平静不是自然状态，是持续努力的结果。越重要的时刻，他的表演越完美。
2. **裂缝在细节里**：他不崩溃。他不失控。他的脆弱体现在那些几乎看不见的细节——多一次呼吸、手腕的一次旋转、半秒的迟疑。这些才是演员和导演需要关注的地方。
3. **[[CH-001]]是他的身份裂缝**：在她面前，他的表演能力下降。这不是缺陷，这是他的成长路径。她看到他"演不下去"的样子，才是他真正学会真实的开始。
4. **他的能量是有限的**：表演完美的代价极高。独处时间的质量决定了他下一次"上场"能撑多久。
