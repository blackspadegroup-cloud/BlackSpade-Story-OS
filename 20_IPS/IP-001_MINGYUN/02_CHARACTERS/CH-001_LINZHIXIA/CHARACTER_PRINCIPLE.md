# CH-001 林知夏 — Character Principle

> 这是 CH-001 林知夏的最高角色原则。
> 以下所有 Layer 均不得违反此原则。

---

**她追逐财富，不是因为爱钱，而是因为贫穷夺走过她的人生。**

---

## 原则解析

| 维度 | 含义 |
|------|------|
| **核心驱动** | 不是贪婪，不是物质主义。是恐惧——对无力感的深层恐惧。 |
| **过去的影响** | 贫穷不是背景故事，是活着的伤口。她曾经因为没钱而失去过重要的东西（人、机会、尊严）。 |
| **财富的意义** | 对她来说，财富 = 安全 = 选择权 = 不再被命运摆布。 |
| **外人误解** | 别人看她觉得她功利、冷血、现实。没人看到她深夜算账时的手在发抖。 |
| **脆弱面** | 如果有人能让她相信"不用那么拼命也可以被爱"，她的整个世界都会动摇。 |
| **成长方向** | 从"用财富对抗恐惧"到"允许自己也被保护"。 |
