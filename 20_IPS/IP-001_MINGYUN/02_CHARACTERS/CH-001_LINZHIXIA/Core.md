# CH-001 林知夏 — Layer 03: Core

## Character DNA（五个核心特质）

### 1. 自立 — Self-Reliant
**定义**：从来不会等别人来救。遇到问题先自己找出路，求人是最靠后的选项。

**压力下的表现**：
- 被逼到绝境时，她反而最冷静。她会把问题拆成可操作的步骤，一个一个去解决
- 如果别人试图帮忙但帮不到位，她会礼貌但干脆地接手：「谢谢，我来吧」——这不是客气，是真的觉得你快不过她
- 极端情况下（比如项目突然崩盘），她会进入一种近乎机械的运转状态：不吃饭、不睡觉、不停打电话，直到事情解决。事后才会虚脱

**这个特质带来的挑战**：
- 无法真正信任团队协作。她会分配任务，但心里始终保留「最后还是要我来兜底」的预期
- 别人觉得她拒人千里——不是不想合作，是她的默认模式就是一个人扛
- 在感情里尤其明显：她不知道怎么接受别人的付出，因为「接受」对她来说等于「欠着」，而她不习惯欠任何人

### 2. 务实 — Pragmatic
**定义**：情感让位于效率。做决定时只问两个问题：有没有用？能不能解决问题？

**压力下的表现**：
- 感情用事的决策者会让她当场皱眉。如果团队里有人因为「感觉」而否定一个数据上合理的方案，她会直接说：「你的理由是什么？数据支持吗？」
- 危机时刻，她是最快做出取舍的人——哪怕那个决定在情感上很难接受

**这个特质带来的挑战**：
- 在需要情感温度的场合（安慰同事、处理客户情绪），她显得机械甚至冷漠
- 对方付出浪漫/感性举动时，她会本能地去分析「对方想要什么」，而不是去感受
- [[CH-002]]的出现会让这个特质受到冲击——因为他是第一个让她发现「有些事情不是用逻辑算得清的」

### 3. 不甘 — Refuses to Lose
**定义**：不是争强好胜，而是绝对不能接受「因为我不够强而失去」。

**压力下的表现**：
- 被人看不起时，她不会当场发作。她会记住，然后用结果说话
- 如果失败了一次，她会复盘到每一个细节，确保同样的问题不会出现第二次
- 这种「不甘」不是对外——所以即使赢了，她也不会去炫耀。她在心里划掉一个目标，然后看下一个

**这个特质带来的挑战**：
- 很难接受自己犯错。犯错对她来说不是学习机会，是「我竟然允许自己犯这种低级错误」的自责
- 对别人也苛刻：她可以理解别人的能力不足，但无法容忍「明明可以做到却因为不够拼而失败」的情况
- 在感情里，如果喜欢上一个人但对方没有回应，她的第一反应不是伤心，而是「我到底哪里不够好？」——她把感情也当成了一场可以靠努力赢下来的局

### 4. 防御性 — Defensive / Walls Up
**定义**：暴露软肋是她最不愿意做的事。所以她在感到脆弱之前，先用冷漠/Sarcasm筑一道墙。

**压力下的表现**：
- 别人靠近她的时候，她下意识往后退半步——不是故意的，是身体记忆
- 被问到私事时会用非常简短的回答挡回去：「还好」「没什么」「就那样」
- 如果被戳到痛处（比如谈及家庭、过去），她会用Sarcasm转移话题：「怎么，采访我的人生故事是要收费的」

**这个特质带来的挑战**：
- 别人觉得她难接近、高冷，也就不会真正试图了解她——这反过来又印证了她的「果然靠不住别人」
- 真正关心她的人需要花很长时间才能穿过她的防御层，而这个过程对她来说也是折磨——因为她一边希望有人看懂她，一边害怕被人看穿
- [[CH-002]]是唯一一个在第一次碰壁后没有退后的人。她对此非常警觉——警觉到花了很长时间才分清楚这到底是他的耐心，还是他的侵略性

### 5. 藏得深 — Emotionally Guarded
**定义**：不是没有感情，是一切感情都被压在水面下。你能看到涟漪，但看不到下面有什么。

**压力下的表现**：
- 即使心里翻江倒海，她表面也是平静的。唯一能看出来的是她叩桌子的频率会变快
- 如果非常难过，她会找一个没有人的地方消化完，然后带着一张没事的脸回来
- 她对母亲的爱很深，但电话里永远只是「吃了」「睡了」「还行」。报喜不报忧是她从十几岁就养成的习惯

**这个特质带来的挑战**：
- 身边的人（包括未来的[[CH-002]]）会觉得她很难「真正在一起」——因为她可以跟你并肩作战，但很难跟你交心
- 长期压抑的结果是：一旦情绪积累到临界点，爆发的方式可能非常剧烈——但她会把爆发也控制在独处时
- 在这个角色线的高潮时刻，她将面临一个选择：继续保持这道墙，或者第一次在另一个人面前把墙拆掉。拆墙对她来说，比跳槽到一个新公司难一万倍

## 核心价值观（按优先级排列）

| 优先级 | 价值观 | 说明 | 违背后果 |
|--------|--------|------|----------|
| 1 | **自立** | 不依赖任何人，不被任何人掌控 | 失去自立 = 失去自我。这是她绝对的底线 |
| 2 | **诚实** | 不说自己不信的话，不承诺做不到的事 | 她可以接受别人骗她（然后让对方付出代价），但绝不允许自己骗人 |
| 3 | **公平** | 付出应该对等，成果应该匹配努力 | 让她最愤怒的事不是失败，而是被人截胡、被抢功、被不公平对待 |
| 4 | **效率** | 时间是最稀缺的资源，不能浪费 | 浪费她时间的人会被她默默标记为「不可靠」——很难翻案 |
| 5 | **忠诚** | 对真正值得的人，她会全力以赴 | 这一条排最后，因为她很少让人进入「值得」的名单 |

## 行为准则（她从不打破的规则）

1. **不欠人情** — 接受别人帮助之前会先问自己：「我还得起吗？」还不起就不要。
2. **不解释** — 她的决定不需要别人同意。如果你问「为什么」，她会给你数据。如果你要的是情感解释，她不会给。
3. **不回头** — 做出的选择不后悔。选错了就承担后果，但绝不在原地后悔。
4. **不展示脆弱** — 在任何人面前。不哭。不诉苦。不抱怨。一句「我好累」都不会说。
5. **不把工作带回家** — 这里的「家」不是物理意义上的办公室——而是指情绪。她可以在家打开电脑加班，但决不让工作中的情绪影响到母亲，也不让任何人看到自己因为工作崩溃。

## 底线（What You Must Never Do to Her）

1. **不要替她做决定** — 即使你是为她好。她这辈子最恨的就是别人替她选择。
2. **不要拿她的出身攻击她** — 说她「从小地方来的」「没有背景」「就是穷怕了」。这是唯一能让她当场翻脸的事——而且是毁灭性翻脸。
3. **不要在她面前贬低她的母亲** — 这是她在这个世界上唯一无条件爱的人。任何人触碰这条线，她会彻底断绝关系，没有任何和解余地。
4. **不要在她建立信任之后背叛她** — 她很少信任人。如果她让你进来了，而你背叛了这份信任——她不会吵，不会闹，但你会永远从她的世界里消失。不是删除，是从来不存在过。
