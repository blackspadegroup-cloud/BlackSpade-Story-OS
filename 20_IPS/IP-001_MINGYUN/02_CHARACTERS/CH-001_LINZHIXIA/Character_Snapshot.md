# CH-001 林知夏 — Character Snapshot

> *阅读时间：30 秒。AI 默认读取入口。*

---

## 一句话介绍

28 岁，投资分析师，从南方小城的贫穷少女爬到大城市的高层办公室——但不管爬多高，她心里那个害怕付不出医药费的小女孩从没离开过。

---

## Character Principle

> 她追逐财富，不是因为爱钱，而是因为贫穷夺走过她的人生。

---

## Character DNA

| 特质 | 说明 |
|------|------|
| **自立** | 没有人会来救她。求助 = 示弱。她一个人扛了太久，忘了可以放下。 |
| **务实** | 感性是奢侈品。她做决定永远先算成本收益——直到有一天她发现有些东西算不出来。 |

| **藏得深** | 真正的她躲在一层又一层的"没事"、"不用"、"我自己来"下面。很少有人见过她不设防的样子。 |

## Current Arc

Season 1 起点：林知夏的生活建立在两个支柱上——工作和存款。她不需要爱情，不需要社交，不需要任何人。

当 [[CH-002|陆景深]] 出现时，她不把他当回事——又一个含着金汤匙出生的少爷，怎么会懂她走过的路。

但陆景深没有被她推开。他看到了她凌晨三点还在办公室的样子，看到了她给母亲转账后对着余额发呆的样子，看到了她在雨中不打伞走路的样子。

他不说什么。只是一次次出现。

她开始害怕了——不是怕他。是怕自己正在期待他出现。

## Current Story Status

项目关键时刻，她被分配与陆景深的团队合作。她准备用专业和冷漠来应对。但她还没准备好的是：陆景深好像看穿了她的专业面具，而且并不打算给她退路。

她以为她最大的挑战是竞争对手。她不知道，最大的挑战是那个笑着问她"你上一次真正开心是什么时候"的人。
