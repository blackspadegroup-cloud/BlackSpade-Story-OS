# CH-001 林知夏 — Emotion Layer

> 她不是没有感情，是把感情锁在了一个只有她自己知道的地方。

---

## 情绪基线

| 维度 | 状态 |
|------|------|
| **日常情绪** | 冷静、警惕、中性偏冷淡。面部表情控制极好，极少泄露真实情绪。 |
| **默认防御姿态** | 随时准备撤退或反击。办公室里听到有人大声说话，她会下意识停顿半秒——评估威胁。 |
| **面部表情控制** | 级别：职业级。面试时不会紧张，谈判时不会露怯，被夸奖时不会脸红。唯一破绽：疲惫到极限时的眼神放空。 |
| **语气特征** | 习惯性平调，极少起伏。讽刺时语调反而更平——这是她的危险信号。 |
| **恢复速度** | 情绪波动后恢复极快。生气不会超过三分钟，悲伤不会在人前停留。不是释然，是习惯性压制。 |

---

## 爱情地图

[[CH-001]] 在感情上几乎是一张白纸——不是因为没有机会，而是因为她从未允许自己进入那个领域。

**核心态度**：爱情是奢侈品，她买不起。不信任，不期待，不靠近。

**具体表现**：
- 工作场合遇到示好，第一时间用专业距离消解所有暧昧可能性
- 好友介绍对象，她礼貌拒绝："我没有时间经营一段关系，对别人不公平"
- 真实原因不是时间不够——是她害怕一旦投入，就会失去自我控制权
- 她曾在日记里写过一句话（从未对任何人提起）："如果一个人爱我，他就有了伤害我的资格。我不想给任何人这个资格。"

**对[[CH-002]]的特殊性**：
- 第一次见到陆景深，她只是觉得"这个人太耀眼，和我是两个世界的人"——这种判断让她安心
- 当陆景深开始展示出对她真实面貌的好奇而非怜悯时，她第一次感到"被看到"的恐慌
- 她对他的感情进展极慢：从"警惕"→"不讨厌"→"好奇"→"依赖"→"害怕"
- 每一步推进都需要她主动压制逃跑的本能

## 友谊地图

| 层面 | 内容 |
|------|------|
| **朋友数量** | 极少。真正能称得上朋友的≤3人。 |
| **交友标准** | 极度严苛：不问她借钱的人、不试探她底线的人、不在她脆弱时假意关心的人。 |
| **忠诚度** | 一旦被她认定为朋友，她可以凌晨三点接电话，可以倾尽所能帮忙。但需要先过她的"观察期"——通常以年为单位。 |
| **友谊中的角色** | 她永远是"提供帮助"的那一方，极少接受帮助。朋友给她送礼物她会焦虑，不知道怎么回报。 |
| **和同事的关系** | 保持礼貌的疏离。不拒绝午餐邀请，但从不主动。不参与办公室八卦，也不透露个人生活。所有人都觉得她"专业但不好接近"。 |

**唯一能让她放下防御的朋友类型**：那种见过她最狼狈的样子，却没有可怜她的人。

## 家庭关系

**母亲** — 这是[[CH-001]]情感版图中最复杂的存在。

| 情感维度 | 内容 |
|---------|------|
| **爱** | 她知道母亲为她付出了一切。母亲一个人打三份工把她养大，从来没有抱怨过。 |
| **怨恨** | 她也怨恨母亲。怨恨母亲的软弱——为什么不去争取？为什么忍受那些屈辱？为什么让她看到那些场面？ |
| **保护欲** | 她工作后第一件事是给母亲买房、买保险、每个月转生活费。她用行动说"我不会让你再受苦"。 |
| **距离感** | 她很少回家，很少打电话。不是因为不爱——是因为每次看到母亲，都会想起小时候的无力感。 |
| **未说的话** | "妈，我不怪你。我怪的是这个世界。但我不知道怎么让你知道我没事。" |

**父亲** — 话题禁区。档案上写"父亲去世"，但她从不对任何人提起具体发生了什么。只知道他在她很小的时候离开了，留下的唯一遗产是"不要靠任何人"这个信念。

## 信任系统

**信任阈值**：极高。不属于"慢慢建立"型，属于"几乎不可能突破"型。

| 信任等级 | 行为表现 |
|---------|---------|
| **Level 0 陌生人** | 礼貌、疏离、信息零泄露。问"周末做什么"会被她用工作话题挡回去。 |
| **Level 1 同事** | 知道她的工作能力和专业背景。不知道她任何私人信息。 |
| **Level 2 熟人** | 知道她单身、住在公司附近、喜欢喝黑咖啡。不知道她的过去、家庭、收入。 |
| **Level 3 朋友** | 知道她小时候家境不好。不知道具体经历过什么。 |
| **Level 4 挚友** | 知道她父亲的事。见过她的脆弱。人数：≤2。 |
| **Level 5 伴侣** | 从未有人到达过这个层级。[ [CH-002] ] 是第一个正在靠近的人。 |

**信任被辜负后的反应**：永不恢复。她不会大吵大闹，不会质问——只会默默把那个人从自己的世界里删除，干净利落，不留痕迹。

## 安全感来源

[[CH-001]] 的 **唯一** 安全感来源：**经济独立**。

| 安全来源 | 重要性 |
|---------|-------|
| **银行存款余额** | ★★★★★ — 她每晚睡前会看一眼账户余额，这是一个仪式。 |
| **可随时变现的资产** | ★★★★★ — 理财、股票、第二收入来源。不能只靠工资。 |
| **职业竞争力** | ★★★★☆ — 知道自己随时能找到下一份工作，这个安全感仅次于存款。 |
| **住房产权** | ★★★★☆ — 租的房子不行。必须是自己的。哪怕很小。 |
| **亲密关系** | ★☆☆☆☆ — 几乎不在这个维度寻找安全感。甚至觉得依赖别人是危险的。 |

**不安感的触发器**：
- 账户余额低于某个心理数字（她设了警戒线，从未告诉任何人）
- 被公司约谈绩效
- 有人提出要"资助"或"帮忙"
- 梦见小时候母亲翻遍口袋找零钱的场景

## 情感表达模式

| 场景 | 真实感受 | 外在表现 |
|------|---------|---------|
| 被欺负时 | 愤怒 + 委屈 | 用冷笑和精准的反击掩盖一切 |
| 被帮助时 | 感激 + 不安 + 觉得欠了人情 | 说"谢谢"语气很硬，然后找机会加倍还回去 |
| 被夸奖时 | 开心 + 不知所措 | "没什么值得夸的" / "只是做了该做的" |
| 被误解时 | 伤心 + 失望 | 直接沉默，不解释——"懂的人不需要解释" |
| 被感动时 | 想哭 + 想逃 | 低头看手机，或者说"我去一下洗手间" |
| 被关心时 | 温暖 + 害怕 | 用讽刺回应："你今天吃错药了？" |
| 极度疲惫时 | 孤独 + 无力 | 在家门口坐五分钟才开门。打开冰箱发呆。 |

**标志性表达方式**：
- 讽刺和冷幽默：这是她的情感语言。如果说的话听起来像在怼人，大概率她在关心。
- 行动代替语言：她不会说"我担心你"，但会默默帮你解决一个问题。
- 沉默：最强烈的情绪表达方式是沉默。生气时不说话，伤心时不说话，爱也不说话。

## 情感盲区

1. **不会撒娇**：即使对人有了依赖需求，她也完全不知道怎么表达。她认为"示弱"是一种缺陷。
2. **不会求助**：宁可自己扛到崩溃，也不开口说"我需要帮助"。她潜意识里认为求助=欠债。
3. **不会表达爱**：她可以说"谢谢"、"辛苦了"、"你很厉害"，但说不出口"我在乎你"、"我需要你"、"我喜欢你"。这些词对她来说太重了，说出来就输了。
4. **低估自己的重要性**：她觉得别人对她的好，总有一天会收回。她不觉得自己值得被长久地爱。

## 情感成长线（Layers 05→Growth）

- **Layer 05 现状**：情感高度防御，安全感单一来源（钱），爱情绝缘体
- **Layer 06 Shadow**：核心伤口让她相信"情感连接=潜在的失去风险"
- **Layer 07 Decision**：情感因素在决策中权重极低
- **Layer 08 Behavior**：一切行为围绕"不被伤害"设计
- **Growth方向**：从"不被伤害"到"允许被爱"，从"独自扛着"到"可以依靠"
