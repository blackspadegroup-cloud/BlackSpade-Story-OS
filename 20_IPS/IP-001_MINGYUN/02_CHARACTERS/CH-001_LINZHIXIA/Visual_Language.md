## Module 13 — Character Visual Language（角色视觉语言）

### Visual Symbol（视觉符号）
黑色橡皮筋（发圈）

### Signature Action（标志性动作）
- 进入工作状态前默默把头发绑起来
- 结束工作或真正放松时放下来

### Signature Object（标志性物件）
黑色橡皮筋，平时戴在手腕上

### Visual Motif（视觉主题）
绑头发 = 工作模式，放头发 = 卸下防备

### Production Rules（制作规则）

| 场景类型 | 头发状态 | 说明 |
|---------|---------|------|
| 工作状态 | 绑头发 | 进入专业模式 |
| 开会 | 绑头发 | 对外呈现干练形象 |
| 提案 | 绑头发 | 高压情境下的自我约束 |
| 熬夜 | 绑头发（松散） | 疲惫但仍强撑 |
| 真正放松 | 放头发 | 与信任的人独处时 |
| 情绪崩溃 | 放头发 | 卸下所有伪装 |
