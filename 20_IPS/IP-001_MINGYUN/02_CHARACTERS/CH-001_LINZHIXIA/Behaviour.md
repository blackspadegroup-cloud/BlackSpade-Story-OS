# CH-001 林知夏 — Behaviour Layer

> 她的身体比她的嘴诚实——只是大部分人都看不懂她的身体语言。

---

## 默认模式

**日常状态**：警觉的冷静。像一只随时准备起跑的猫——表面上看起来很放松，实际上每一块肌肉都是待命状态。

**职业模式**：
- 走进会议室时，先扫视全场——评估在场的人、位置、权力动态
- 坐姿端正但不僵硬，双手放在桌面上（不遮挡、不防御，这是她刻意训练的）
- 发言前会停顿0.5秒——整理语言，确保没有漏洞
- 被提问时从不立即回答，先沉默两秒

**非职业模式**（独处时）：
- 肩膀塌下来，后背微微弯曲
- 不自觉地按太阳穴
- 长时间盯着一个方向发呆
- 手机不离手——不是在看，是握着才有安全感

## 常见习惯

| 习惯 | 频率 | 说明 |
|------|------|------|
| **睡前查账户** | 每晚 | 确保余额在安全线以上。低于安全线→失眠。 |
| **提前15分钟到达所有约定** | 100% | 从不迟到。迟到=对事情失去控制。 |
| **列计划表** | 每天 | 精确到小时。如果计划被打乱→焦虑，但不会表现出来。 |
| **喝黑咖啡** | 每天3-4杯 | 不加糖不加奶。不是因为喜欢——是因为"效率最高"。 |
| **整理桌面** | 每天下班前 | 离开办公室前必须把所有东西归位。乱=失控。 |
| **算优惠** | 每次消费 | 即使用得起，也会下意识算性价比。买贵的东西需要做三天的心理建设。 |
| **删聊天记录** | 定期 | 不想被人看到自己的对话。不是有秘密——是不想留下痕迹。 |
| **抿嘴** | 紧张/压力时 | 下唇往里收，嘴角绷紧——是她最明显的微表情。 |
| **看手表/手机时间** | 频繁 | 确认时间。确认一切在掌控之中。 |
| **保存所有收据和发票** | 习惯性 | 以备万一。她心里有一个"万一"列表，大部分根本不会发生。 |

## 身体语言

### 职业环境中的身体语言

| 部位 | 姿态 | 含义 |
|------|------|------|
| **肩膀** | 微微后展，胸腔打开 | 有意识地制造自信姿态（后天习得，非自然） |
| **手臂** | 极少交叉。放在桌面或自然下垂 | 不想显得防御（但她内心其实是紧张的） |
| **手指** | 说话时用手指轻叩桌面/键盘边缘 | 思考节奏。叩越快→越不耐烦 |
| **眼神** | 直視对方，但不会一直盯着 | 控制性对视——不想输，也不想太有攻击性 |
| **头部** | 微微倾斜 | 她在认真听，但也在评估 |
| **嘴唇** | 自然闭合，不笑 | 不是不开心——是觉得工作中不需要展示情绪 |

### 非职业环境中的身体语言

| 部位 | 姿态 | 含义 |
|------|------|------|
| **肩膀** | 微微内收 | 放松→但仍然有一点防御 |
| **手臂** | 抱胸/拿着手机/握着杯子 | 需要手里有东西作为屏障 |
| **手指** | 无意识地抠指甲边缘 | 焦虑或不安 |
| **眼神** | 容易飘开 | 面对情感话题时无法直视对方 |
| **坐姿** | 缩在椅子角落 | 下意识占据最小空间 |
| **走路** | 低头、快步 | 不想被人注意到 |

### 和[[CH-002]]在一起时的特殊身体语言

- 无法长时间直视他的眼睛——每次对视都会先移开视线
- 如果他不小心碰到她→身体会僵住0.5秒，然后自然退半步
- 但当他说话时，她会不自觉地微微靠向他那边
- 嘴角会有一个极其微小、几乎不可见的弧度——那不是笑，是"我在你身边放松了一点点"
- 这个微表情只有最熟悉她的人才看得出来

## 压力反应表

| 压力源 | 即时反应 | 持续反应 | 外部可见信号 |
|--------|---------|---------|------------|
| **工作截止日期** | 深吸一口气，然后加速 | 减少睡眠、取消社交 | 眼圈变黑、咖啡杯变多 |
| **财务压力** | 打开银行APP查余额 | 降低所有非必要开支 | 消费更谨慎、拒绝聚餐 |
| **人际冲突** | 先沉默，再理性回应 | 和该人保持距离、减少接触 | 语气变平、笑容消失 |
| **被批评** | 面部无表情→內心快速复盘 | 用行动证明对方错了 | 工作更拼命、更少说话 |
| **被忽视/被怠慢** | 无所谓的外表 | 内心记一笔，观察模式 | 态度礼貌但更疏离 |
| **情感波动（关于[[CH-002]]）** | 心跳加速→面部控制 | 回避他/过度专注于工作 | 加班到很晚、回复信息变慢 |
| **创伤触发**（想到童年） | 突然的静默 | 自我封闭2-3天 | 不回消息、午休不出现、眼神空洞 |
| **身体不适/生病** | 假装没事 | 直到症状严重影响工作才请假 | 面色差但说"没事" |

**极端压力下的崩溃表现**：
- 不会哭。在公众场合不会。她崩溃的方式是——停止说话，眼睛望着某处，然后过一会儿继续做事
- 如果真的要哭，她会去卫生间锁上门，等眼睛不红了再出来
- **唯一一次真正的崩溃**（剧本关键场景）可能是她母亲出事/或[[CH-002]]出事时——她会先冷静处理一切，然后在事情解决后、所有人都走了之后，一个人蹲在某个角落哭

## 喜好体系

### 喜欢

| 类别 | 具体 | 原因 |
|------|------|------|
| **饮品** | 黑咖啡（深烘）、热白开水 | 效率、健康，不需要为味道花钱 |
| **食物** | 简单的家常菜：番茄炒蛋、清蒸鱼 | 小时候母亲的拿手菜，有安全感 |
| **衣物** | 基础款、中性色、好面料 | 百搭、不过时、不需要频繁购置 |
| **居住** | 小而整洁、有阳台、光线好 | 可以一个人待着，有阳光就不会觉得太冷清 |
| **活动** | 看书、听古典音乐、一个人散步 | 不需要和别人互动，不用说话 |
| **时间** | 清晨6:00-8:00 | 全世界还没醒，所有时间都是她的 |
| **书籍** | 非虚构类：心理学、社科、人物传记 | 了解人、了解规则、了解如何生存 |
| **状态** | 日历上没有任何安排的一天 | 没有义务、没有社交、没有表演 |

### 不喜欢

| 类别 | 具体 | 原因 |
|------|------|------|
| **食品** | 甜的、腻的、花哨的甜品 | 本能排斥"不实用"的消费 |
| **社交** | 团建、酒局、陌生聚会 | 消耗大量能量，没有实质产出 |
| **行为** | 迟到、爽约、临时变卦 | 不尊重别人的时间=不尊重人 |
| **消费** | 奢侈品logo、网红打卡、为面子买单 | "把钱花给别人看，是最愚蠢的事" |
| **话题** | 被人问收入、家庭、感情状态 | 隐私边界被侵犯 |
| **人** | 炫富的、假热心的、没边界感的 | 触发她的防御机制 |
| **天气** | 阴雨天持续超过三天 | 会影响情绪（虽然她不承认） |
| **节日** | 春节、情人节、任何需要"表达情感"的节日 | 不知道该怎么表现，压力很大 |

## 行为触发器

### 触发→防御行为

| 触发条件 | 自动反应 | 行为表现 |
|---------|---------|---------|
| 感到被怜悯 | 刺猬模式启动 | 语气变硬，用讽刺回击 |
| 感到被施舍 | 拒绝+逃离 | "不用了，谢谢"（语气冷到结冰） |
| 感到被看穿 | 情绪撤退 | 沉默、转移话题、离开现场 |
| 感到被需要（情感上） | 不知所措 | 用工作/实用帮助代替情感回应 |
| 感到被亲近 | 轻微的后退 | 身体向后倾、眼神移开 |

### 触发→正向行为

| 触发条件 | 自动反应 | 行为表现 |
|---------|---------|---------|
| 看到有人被不公平对待 | 正义感被激活 | 冷静介入，用制度和逻辑说话 |
| 看到有人需要实际帮助 | 行动模式启动 | 默默地提供帮助，然后装作没做过 |
| 被真诚地尊重 | 防御降低5% | 话会多一点点，语气缓和一点点 |
| 和[[CH-002]]单独相处 | 警惕+不自觉地软化 | 语速变慢，偶尔会有极淡的笑意 |

## 日常时间切片（行为写实）

**工作日**：
```
06:00  起床。看手机。查账户余额。喝一杯白开水。
06:30  跑步/瑜伽（如果前天没加班太晚）。
07:00  洗澡。黑咖啡。检查今天的会议安排。
07:45  出门。提前15分钟到办公室。
08:00-12:00  专注工作。不看手机。
12:00  午餐。如果一个人——边吃边看行业报告。
13:00-18:00  会议+项目工作。咖啡第二杯、第三杯。
18:00-20:00  如果有加班→加班。如果不加班→回家。
20:00-22:00  家里。做饭（简单的那种）。看书/看工作资料。整理明天计划。
22:00-23:00  洗漱。关手机。睡前最后一次查账户。
23:00  关灯。躺着。睡不着也想事情。想累了就睡着了。
```

**周末**：
```
- 如果没事→睡到自然醒（8:00左右，也算她的"自然醒"）
- 洗衣服、整理房间、采购一周所需
- 可能有半天在看专业书/准备证书考试
- 晚上可能会一个人去超市慢慢逛（这是她为数不多的放松方式）
- 社交活动：几乎为零。偶尔和那2-3个朋友吃顿饭。
```

**一个反常但真实的行为细节**：
她每隔一段时间会去银行ATM，不是为了取钱。就是把卡插进去，查一下余额，看一眼那个数字，再退出来。数字在那里，她就安心了。

如果某天她发现余额低于心理安全线，她会：
1. 在那周取消一切非必要支出
2. 不吃外卖，自己做饭带便当
3. 晚上多接一个私活/多看一份兼职资料
4. 直到余额回到安全线以上，才恢复正常

这个行为模式，没有人知道——包括她的母亲。
