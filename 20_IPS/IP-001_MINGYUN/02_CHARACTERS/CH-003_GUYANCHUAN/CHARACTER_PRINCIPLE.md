# CH-003 顾言川 — Character Principle

> 这是 CH-003 顾言川的最高角色原则。
> 以下所有 Layer 均不得违反此原则。

---

**他拼命追求自由，却第一次心甘情愿，为一个人停下脚步。**

**核心命题：** 自由（Freedom）

---

## 原则解析

| 维度 | 含义 |
|------|------|
| **核心驱动** | 不是爱玩，不是不负责任。是恐惧——对束缚的深层恐惧。 |
| **自由的真相** | 对他来说，自由 = 安全。不承诺 = 不受伤。不扎根 = 不需要面对失去。 |
| **外在表现** | 所有人都觉得他潇洒、随性、活得通透。没人注意到他从来没有在一个地方停留超过两年。 |
| **内在真相** | 他的自由不是追求——是逃跑。害怕的不是安定，是安定之后可能发生的背叛、冷漠、重复父母的剧本。 |
| **脆弱面** | 林知夏第一次让他觉得——也许留下，比离开更需要勇气。他害怕的不是感情，是发现自己竟然想为一个人停下来。 |
| **核心矛盾** | 他渴望深度连接，但恐惧承诺。他想要真实的亲密，但从小看到亲密如何变成交易。 |
| **成长方向** | 从"自由 = 没有束缚"到"真正的自由 = 有能力选择留下"。 |

---

## Signature

**相机。**

意义：相机是他的眼睛，也是他的盾牌。透过取景器看世界，比直接面对容易得多。

当他感到不自在、被看穿、或情绪即将失控时，他会举起相机——既是创作，也是逃避。

---

## Core Arc

```
追求自由 → 逃避承诺 → 遇见林知夏 → 
第一次停下脚步 → 学会承担责任 → 
真正理解爱的意义 → 
发现真正的自由来自选择，而不是逃避
```
