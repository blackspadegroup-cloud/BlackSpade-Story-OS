# Relationship Engine — 顾言川

## 关系形成模式

### 第一阶段：轻连接（所有人）
顾言川交朋友的能力几乎是超自然的。他可以在安检队伍里交到一个朋友，在街角咖啡店认识一个十年的好友。

**方法**：
- 用相机破冰——"你这个角度很好看，介意我拍一张吗？"
- 用自嘲降低威胁感——"我看起来像骗子？那——我请你喝杯咖啡证明我不是。"
- 用故事建立连接——他有一万个有趣的故事，每个都精心挑选过，安全、好听、不涉及他自己

### 第二阶段：中度连接（少数人）
有人走进了重复交往的圈。他记得你的喜好，从机场寄明信片，看到相关的东西会发给你。

**但依然存在天花板**：
- 他知道你的一切
- 你知道他有趣的故事
- 但你不知道**他**

### 第三阶段：深度连接（几乎无人）
他的防御系统在这里启动。除非有人：
1. 不把他的幽默当全部
2. 在他转移话题时不让他转
3. 看到了他的沉默，不离开

## 关系维护模式

### 轻维护 ⭐⭐⭐⭐⭐
- 明信片。每个城市一张。简短、随意、温暖
- 机场消息——"登机了，回来聚"
- 随手拍的照片——"这光，想到你"
- 节日/生日从不忘记

### 深维护 ⭐
他不会主动发起深度对话。
不会第一个说"我想你"。
不会在深夜打电话说"我需要你"。
**不是不愿意。是不会。**

## 依恋类型

**恐惧型回避（Fearful-Avoidant）**

| 模式 | 表现 |
|------|------|
| 渴望连接 | "他一直在人群里。他需要人群。" |
| 恐惧深入 | 一旦有人靠近真正的他——他找借口离开"去拍照" |
| 不信任持久 | 潜意识认为：好的东西都会结束。所以不如不要开始 |
| 自我 sabotage | 当一段关系变得重要——他会做点什么来保持距离 |
| 测试行为 | 在潜意识里测试对方会不会离开 |

> **内心独白（他永远不会说出口的）**：
> "如果我停下来，如果我认真了……然后呢？然后她发现我不是她以为的那个人。然后她走。那我还不如先走。"

## 社交电池

**Ambivert — 混合型**

| 状态 | 续航 | 恢复方式 |
|------|------|----------|
| 社交模式 | 很强 — 可以在派对上活跃一整晚 | 但第二天需要一个人待着 |
| 深度社交 | 有限 — 一次深刻的对话后需要逃离 | 一个人去拍照，去陌生的街道 |
| 人群后 | 电量会突然归零 | 他会消失几天 |

> **标志性行为**：派对结束后，他不和任何人一起走。他一个人走回住处。走很长的路。

## 群体角色

他是**群体中的轻型力量**。

- 他是讲故事的人
- 他是让新人不感到尴尬的人
- 他是调节气氛的人
- 他是那个——每个人都喜欢，但没有人真正了解的人

**他不在时**：
- 大家会问："顾言川呢？"
- 但没有人能真正回答他去哪儿了

## 权力动态

**厌恶层级结构。**

- 不喜欢在别人之上：不想被崇拜，不想负责任
- 不喜欢在别人之下：不想被指挥，不想被约束
- 偏好：**水平关系**。大家都是平等的

**与 [[CH-002]] 陆景深的关系**：
- 他与陆景深天生互为镜像
- 陆景深的存在让顾言川本能地想逃——因为陆代表了责任、束缚、秩序
- 但顾言川也是唯一能让陆景深放松的人
- 他们的对话模式：顾言川用笑话打破陆的沉闷，陆用沉默让顾言川无处可躲

## 与 [[CH-001]] 林知夏的特殊性

### 她的不同之处
林知夏是第一个**不被他幽默骗过**的人。

- 别人听他讲笑话——她看他讲笑话时的眼睛
- 别人觉得他开心——她问"你还好吗"
- 别人说"你好有趣"——她说"你不用每次都这么有趣"

### 这对他的影响
他第一次不知道该怎么应对一个人。

- 他的防御系统（幽默）在她面前失效了
- 他发现自己在说真话——然后恐慌
- 他发现自己想留下——然后更恐慌

### 关系的推进模式

| 阶段 | 顾言川的行为 | 内心的真实 |
|------|-------------|-----------|
| 初期 | 对她讲更多笑话 | "她不一样。但这不可能。" |
| 意识 | 开始不自觉地为她改行程 | "我只是……刚好想去同样的地方。" |
| 恐惧 | 突然冷淡几天 | "我跑远一点，这种感觉就会消失。" |
| 抉择 | 第一次取消机票 | "操。我是认真的。" |
| 承诺 | 不再找借口 | "我不走不是因为你绑住了我。是因为我选了你。" |

## 关键关系动态

### 与家庭
- 不说话。但没切断
- 每年寄一张明信片回家——没有回信地址
- 他恨的不是父母，是那个家教会他的东西：爱情会死，婚姻是牢笼，自由比什么都重要

### 与同行 / 摄影圈
- 受尊敬，但保持距离
- 合作过的人都说他好相处
- 但没人真的了解他

### 与过去的恋人
- 分开时都是和平的
- 没有人恨他
- 但也没有人真正走进过他
- 每段关系结束时，对方会说："我觉得你从来没让我真的靠近你。"
- 顾言川听到这句话，会笑着说："大概吧。"

> **核心矛盾**：
> 他擅长一切关系的**开始**。
> 他害怕一切关系的**深处**。
> [[CH-001]] 是第一个让他想试试看的人。
