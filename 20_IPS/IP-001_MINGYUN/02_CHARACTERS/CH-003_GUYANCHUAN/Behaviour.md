# Behaviour Layer — 顾言川 (CH-003)

> *"注意他什么时候举起相机。那才是他真正说话的时候。"*

---

## 默认模式

**日常状态**：放松的，带着一丝似有若无的笑意。像一只在阳光下打盹的猫——看起来很慵懒，但眼睛里什么都在看。

**社交状态**：温暖的、有趣的、让人如沐春风。他是那种能让陌生人十分钟内说出自己故事的人。

**独处状态**：安静，眼神会变深。会反复看相机里的照片，手指摩挲相机包的织带。

---

## 常见习惯

### 标志性动作：相机抬起

这是顾言川最重要的行为标记。无论是数码还是胶片，那台徕卡M系列几乎是他身体的延伸。

**相机抬起的含义谱系**：

| 情境 | 含义 |
|------|------|
| 进入一个新的空间 | "我先观察" |
| 有人开始谈论深刻话题 | "我要退一步" |
| 他感到紧张/害羞 | "让我躲一下" |
| 看到动人的画面 | "我想记住这个" |
| 面对 [[CH-001]] 的真实情绪 | "我还没准备好面对" |

**进阶观察**：当他真的被触动时，举起相机的动作会慢半拍——那一刻他在挣扎：是留下面对这个瞬间，还是通过镜头逃避它。

---

### 日常习惯清单

- **摸相机 strap**：思考或不自在时，手指会无意识地去摸相机背带的边缘
- **拍细节**：不拍宏大的东西——他拍酒杯边的水痕、剥落的墙皮、陌生人手部的姿势
- **用幽默防守**：当话题转向他自己时，会突然说一句好玩的话"转移视线"
- **躲避眼神**：说真话的时候反而不看人——假装在看构图、在看光线、在看任何别的地方
- **坐不住**：同一个姿势不会保持太久。喜欢窗边座位——为了能看到外面
- **翻看照片**：安静的时候会翻相机里的照片，像是在确认自己真的经历过那些地方
- **记名字**：见过一次的人他都能记住名字——这是他的一种温柔，也是一种距离控制

## 身体语言

| 情境 | 身体语言 |
|------|----------|
| 轻松/社交 | 身体打开，手势多，喜欢用手比划讲故事 |
| 紧张/回避 | 肩膀微微前倾，手指活动增多（摸相机、转笔） |
| 专注/拍摄 | 几乎静止——只有手指在调焦，呼吸变浅 |
| 真正的情感时刻 | 身体会细微地僵住，然后——相机抬起来 |
| 被看穿 | 嘴角的笑凝固一秒，然后笑得更大了 |
| 决定留下 | 肩膀落下——第一次看到他真正放松的样子 |

**关于空间**：他习惯与人保持一个手臂的距离。不是排斥，是本能。如果 [[CH-001]] 跨过这个距离而他没有后退——那比任何情话都重要。

## 压力反应模式

| 压力等级 | 行为表现 | 可观察信号 |
|----------|----------|-----------|
| **轻度** | 开玩笑增多，话题跳跃 | 说话比平时快，笑点低 |
| **中度** | 坐立不安，找借口走动 | "我去买杯咖啡"、"我去拍个东西" |
| **重度** | 突然沉默，然后离开 | 不再笑了，直接走——不解释 |
| **危机级** | 冷静、专注、行动导向 | 切换为"导演模式"——这是最可靠的他 |

**压力下的恢复行为**：一个人待着，翻照片，然后去人多的街头走一圈。用陌生人的喧嚣洗掉自己的情绪。

| 情绪 | 外部行为 | 他人看到的 | 实际内心 |
|------|----------|-----------|----------|
| **悲伤** | 开玩笑 | "他今天格外幽默" | 心底在下雨 |
| **愤怒** | 沉默或离开 | "他突然有事走了" | 怕自己说出伤人的话 |

| **真正的高兴** | 安静地笑 | 那种没有声音的笑 | 这是最真实的时刻 |
| **感动** | 相机抬到一半，又放下 | 犹豫了一下 | 他选择真实地面对这一刻 |
| **恐惧（关于感情）** | 过度保护 | "他今天好贴心" | 他在用温柔掩饰害怕 |
| **嫉妒** | 故意表现得毫不在意 | "啊，挺好的" | 笑得最自然的时候最在意 |

## 喜欢

- **新的地方**：任何他没去过的城市、没走过的巷子
- **陌生人的故事**：路边摊老奶奶的人生、火车上背包客的经历
- **早晨的光**：清晨六点的光线，斜斜的，金黄色的
- **胶片颗粒**：数码太干净了，他喜欢不完美的质感
- **街头食物**：越是当地人的小摊越好——他拍，也吃
- **爵士乐**：尤其是那种有点走调的萨克斯风
- **旧皮面笔记本**：他有一本五年前的，封面已经磨得发亮
- **雨天的窗**：咖啡馆靠窗的位置，雨打在玻璃上

---

## 不喜欢

- **西装**：尤其是领带——"脖子上套根绳子"是他的原话
- **会议室**：四面墙，没有窗，长桌——"这不是房间，是盒子"
- **"你五年计划是什么？"**：这是他最讨厌的问题，没有之一
- **家庭晚宴**：不是不想见妈妈，是不想在那个冰冷的饭厅里坐着
- **在一个城市超过三个月**：三个月是阈值——之后就会开始痒
- **被人追问感情**："你和那个女孩……""——相机抬起来，人走了"
- **承诺这个词**：比起"我爱你"，他更怕"我答应你"

---

## 日常场景行为

### 早晨
睡得很浅，天亮就醒。第一件事不是看手机，是看窗外——判断今天的光线。如果没有拍摄，他会带着相机出门散步，去一个没去过的早餐店。

### 社交场合
前半段是他最热闹的时候——认人、搭话、拍合影。后半段他会慢慢退到角落，或者提前消失。"他还在吗？""哦，他说去拍夕阳了。"

### 旅行中
在机场或车站是他最自在的时候。脚步轻快，表情比在任何地方都松弛。他会拍下每一个到达大厅的时钟——像在标记"我在这里"。

### 面对 [[CH-001]]
- 她说话时，她会听——真正的听
- 但当她看着他的眼睛，他的视线会飘走
- 他会记得她所有的小习惯，但他不会承认
- 他会偷偷拍她——相机放在桌上"不小心"按到快门那种
- 当她需要他，他会在，但他会说"刚好路过"

## 声音与语言特征

- **音色**：偏低沉的暖嗓，放松时有微微的沙哑
- **语速**：正常偏快，说到有趣的事情时会加快
- **口癖**："没事没事"、"差不多"、"再说吧"
- **语言习惯**：
  - 喜欢用问句代替陈述——"你不觉得吗？"、"是吧？"
  - 很少用"永远""一定"这种绝对词
  - 说到自己时，主语经常模糊——不是"我不想"，是"走不走都行"
  - 说到别人的故事时，语气会变得特别生动

## 行为弧光：从逃跑到留下

| 阶段 | 行为特征 | 可观察线索 |
|------|----------|-----------|
| **初期** | 轻松、有趣、保持距离 | 相机永远在手上，笑永远在脸上 |
| **拉扯期** | 忽近忽远，冰火交替 | 更用力地开玩笑，但沉默的时间也变长了 |
| **临界点** | 相机举到一半，放下了 | 他选择了面对，而不是记录 |
| **选择期** | 开始出现"停留"的行为——买了用不上的东西，开始整理空间 | 行李箱不再一直敞着 |
| **安定期** | 肩膀第一次真正放松下来 | 他能坐在同一个地方，不说话了，也不觉得需要逃走 |
