# CH-003 顾言川 — 角色快照

> **一句话介绍：** 他是个永远在路上的男人——不是因为爱流浪，是因为怕停下来之后发现没有值得留下的东西。直到遇见了一个让他开始怀疑「下一站」意义的女人。

**原则：** 他拼命追求自由，却第一次心甘情愿，为一个人停下脚步。
**核心主题：** 自由（Freedom）
**DNA：** 自由 · 温暖 · 逃避 · 敏锐 · 藏伤

---

## 一、核心驱动力（Core Drive）

| 维度 | 内容 |
|------|------|
| **Want（渴望）** | 真正的自由——不被束缚、不被期待、不被任何人拴住。想走就走，想停就停。 |
| **Need（需要）** | 学会留下——真正的自由不是「随时可以离开」，而是「有能力选择留下」。他需要的不是一个人放他走，是一个人让他想留。 |
| **Wound（创伤）** | 出生在商业联姻家庭，父母之间没有爱，只有交易和冷暴力。父亲告诉他「不要活成他」，母亲留下一句「你长大了，不需要我了」然后远去。他学会的第一件事：不要依赖任何人。 |
| **Lie（谎言）** | 「我不需要任何人。我一个人的生活就很完整。」 |
| **Truth（真相）** | 他不是不需要——他是不敢需要。因为他见过最亲密的人之间可以变成什么样子，所以他选择从来不靠近。 |

## 二、当前弧线（Current Arc）

### Season 1：初遇 — 第一次想留下

**起点：** 顾言川刚从一次跨洲纪录片拍摄回来，被朋友拉去一个商业酒会。他打算待十分钟就走——然后在角落里看到了一个同样想走的女人。

**弧线描述：**
- 第一阶段：他觉得她有趣——一个在商业酒会上不社交、不微笑、看起来准备落荒而逃的法律顾问。他决定救她出去，作为今晚唯一的善举
- 第二阶段：他开始发现她不只是「有趣」——她是真实的。她在他说笑话时会真正笑出来，在他递上咖啡时会说谢谢而不是假装客气。他发现自己开始想多待一会儿
- 第三阶段：他发现自己开始取消行程、推迟出发、翻看手机等她消息——他做了所有以前绝对不会做的事
- 弧线高潮：他说出了那句连自己都没准备好要听的话——「你知道最可怕的是什么吗？我发现我不怕走了——我怕走不了。」

**当前状态：** S1 中段。他正在经历人生中第一次「想留下」的冲动，但还没有完全承认。他从「一个有趣的陌生人」开始，逐渐变成「一个他愿意为她改签机票的人」。
