# 示例角色 — Character Snapshot

---

### 一句话介绍

表面是优雅从容的画廊主理人，实则背负着 17 年前未能拯救挚友的创伤，在追查真相与保护现有生活之间挣扎。

---

### Character DNA

- **不可动摇的正义感** — 即使代价是自己的一切
- **过度自省** — 永远先检讨自己哪里做错了
- **守护者本能** — 无法对身处困境的人袖手旁观
- **情感防御机制** — 用冷静和幽默掩盖内心脆弱
- **敏锐的观察力** — 能注意到最细微的异常，但选择性忽略不愿面对的线索

---

### Core Drive

| 维度 | 内容 |
|------|------|
| **Want（想要）** | 找到 17 年前好友失踪的真相，让逝者安息 |
| **Need（需要）** | 学会原谅自己的无能为力，放下「我必须拯救所有人」的执念 |
| **Core Wound（核心创伤）** | 17 年前亲眼目睹好友被带走却未能呼救，从此活在无声的自责中 |
| **The Lie（谎言）** | 「只要我足够努力、足够谨慎，我就能防止任何悲剧发生」 |
| **The Truth（真相）** | 有些事不在掌控之中；真正的勇气不是防止一切伤害，而是在伤害发生后仍然选择相信和前行 |
