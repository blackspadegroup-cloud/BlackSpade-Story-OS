# BSOS 创作原则

> 本文件是对创作宪法的逐条展开阐述，包含具体应用示例和判断标准。

---

## 原则一：揭示优于解释（Show, Don't Tell — Extreme Mode）

### 宪法依据
宪法第一条：Never Explain. Always Reveal.

### 展开阐述
信息渗透必须经过角色的行为滤镜。读者通过观察角色做了什么来理解角色，而不是被告知角色是什么。

### 判断标准
| 违规行为 | 正确做法 |
|---------|---------|
| 作者旁白直接描述性格 | 通过角色在压力下的选择来展现 |
| 内心独白解释动机 | 用角色冲突中的行动暗示动机 |
| 对话中角色直接说出真相 | 用细节碎片让读者自行拼凑 |

### 示例
**场景：揭示主角是退伍特种兵**

- ❌ 旁白式：**"李明曾是一名特种部队成员，拥有丰富的战斗经验。"**
- ✅ 揭示式：超市抢劫案发生时，李明在众人尖叫抱头时第一时间扫视出口、评估劫匪站位、用眼神示意收银员蹲下——下意识动作暴露了训练背景。

### 越界警告
当读者需要外部解读才能理解剧情时，说明揭示不够充分。

## 原则二：角色驱动剧情

### 宪法依据
宪法第二条：人物创造剧情，不是剧情创造人物。

### 展开阐述
剧情是角色选择的自然结果。如果从剧情倒推角色行为，就违反了这一原则。一个好的测试方法是：把角色换掉，剧情还能成立吗？如果能，说明剧情不是角色驱动的。

### 应用指南
1. **建立角色核心动机清单**：每个主要角色必须有清晰的"想要什么"和"恐惧什么"
2. **选择即角色**：关键剧情节点上，角色做出的选择必须反映其核心价值观
3. **不允许剧情盾**：角色不能"突然变聪明/变勇敢/变愚蠢"来推动剧情

### 示例
- ✅ 一个渴望复仇的角色闯入敌营（角色动机驱动剧情）
- ❌ 剧情需要一场战斗，所以温和的角色突然主动挑衅（剧情驱动角色）

## 原则三：成长是真正的主线

### 宪法依据
宪法第三条：爱情只是表层，成长才是真正主线。

### 展开阐述
感情线（爱情、友情、亲情）是角色成长的载体和映射，而不是终极目标。角色通过人际关系获得的成长，才是故事的核心价值。

### 应用示例
**爱情线处理方案对比：**

| 维度 | ❌ 表层处理 | ✅ 成长处理 |
|------|-----------|-----------|
| 核心 | "他们能否在一起？" | "这段关系如何改变他们？" |
| 冲突 | 外部阻挠（家长、阶级） | 内在矛盾（恐惧、价值冲突） |
| 结局 | 在一起/不在一起 | 彼此成就/彼此毁灭/互相超越 |

### 检查清单
- [ ] 如果把所有感情线删除，故事还剩什么？
- [ ] 角色在关系结束时，是否变成了不同的人？
- [ ] 感情的发展是否与角色的成长弧同步？

## 原则四：命运的贯穿与伏笔管理

### 宪法依据
宪法第四条：命运（Fate）必须贯穿整个故事，所有伏笔都必须有回收。

### 伏笔分类法
| 类型 | 定义 | 最大保留时限 |
|------|-----|------------|
| 短期伏笔 | 本章或下一章内回收 | 2 章节 |
| 中期伏笔 | 弧内回收 | 1 卷 |
| 长期伏笔 | 跨卷回收 | 整部作品 |
| 系列伏笔 | 跨部作品回收 | 系列终章 |

### 伏笔管理规则
1. **设立伏笔追踪表**：每个伏笔必须有 ID、埋设位置、预计回收位置、状态
2. **允许误导线**（Red Herring），但误导线本身也属于伏笔，必须在某处揭穿
3. **遗忘即违规**：任何超过最大保留时限仍未回收的伏笔视为创作事故

### 命运主题的贯彻
命运不是简单的"冥冥中注定"，而是角色在看似自由的选择中，一步步走向无法逃脱的终点，或是通过对抗命运完成了命运本身。

## 原则五：角色自主性保障

### 宪法依据
宪法第五条：没有工具人角色，所有角色都必须有目标。

### 角色深度分级

| 级别 | 定义 | 标准 | 适用范围 |
|------|-----|------|---------|
| L1 - 背景 | 有名字，无目标 | ❌ 不允许 | — |
| L2 - 功能 | 有目标，服务于剧情 | 最低可接受 | 次要配角 |
| L3 - 完整 | 有目标+弧线 | 推荐标准 | 主要配角 |
| L4 - 驱动 | 目标推动剧情 | 必须 | 主角/重要角色 |

### "工具人检测" 快速测试
- 如果这个角色消失，剧情能否照常推进？
- 这个角色除了帮主角做某件事，还有自己的故事吗？
- 这个角色的愿望和恐惧是什么？

三个问题中如果全部答不上来，该角色就是工具人，需要重写。

## 原则六：叙事节奏阶梯

### 宪法依据
宪法第六条：每一集必须有 Hook，每三集必须有一次高潮。

### Hook 类型
| 类型 | 示例 | 效果 |
|------|------|------|
| 悬念型 | "门突然被推开，进来的是..." | 好奇心驱动 |
| 反转型 | "他摘下面具，露出了那张不可能的脸。" | 认知颠覆 |
| 情感型 | "她说的最后一句话是：我从来没有爱过你。" | 情感冲击 |
| 决策型 | "枪口瞄准了那个人的后背。扣还是不扣？" | 期待选择 |

### 高潮层级
- **小高潮**（每卷）：该卷核心冲突的顶点
- **中高潮**（每 3 集）：剧情弧的转折点
- **大高潮**（每卷结尾）：推动主线发生质变的事件

### 节奏检查
每 3 集标记在 Story Map 上，确认高潮点的存在和强度。

## 原则七：谜团闭合义务

### 宪法依据
宪法第七条：每一个谜团最终必须有答案。

### 谜团生命周期
```
谜团提出 → 线索积累 → 读者猜测 → 答案揭晓 → 影响释放
```

### 谜团分类处理
| 类型 | 例子 | 闭合要求 |
|------|------|---------|
| 核心谜团 | 主角的身世 | 必须在主线结束前闭合 |
| 支线谜团 | 某个配角的秘密 | 可以不闭合，但必须标识 |
| 开放性谜团 | 宇宙的终极问题 | 可以留白，但须有回应 |

### "谜团标识" 规则
- 若某个谜团在本作中不打算解答，必须在文本中明确标记（如角色说"这个秘密就让时间来揭晓吧"）
- 未标记的未闭合谜团视为创作事故

## 原则八：反转的内在同一性

### 宪法依据
宪法第八条：不要为了反转而反转，所有反转必须符合人物性格。

### 反转质量矩阵
| 质量 | 读者第一反应 | 读者重读反应 |
|------|------------|------------|
| 优质反转 | "天哪！" | "原来如此，早有铺垫！" |
| 平庸反转 | "哦。" | "好像也没别的可能。" |
| 劣质反转 | "什么鬼！" | "完全不合理。" |

### 反转的合法性检查
1. **铺垫追溯**：反转的线索是否在前文中存在？
2. **性格一致性**：反转后的行为是否符合角色核心动机？
3. **因果链**：是否有合理的因果链条导致反转发生？
4. **必要性测试**：去掉这个反转，故事是否更好？

## 原则九：Character DNA 维护

### 宪法依据
宪法第九条：所有人物行为必须符合 Character DNA。

### Character DNA 定义框架
每个角色文档必须包含以下 DNA 要素：

```
角色: [名称]
DNA 编码:
  核心价值观: [x, y, z]     # 不可违背
  核心恐惧: [a, b]           # 驱动逃避行为
  行为模式: [m, n]           # 习惯性反应
  成长方向: [p ← 现状 → q]   # 可能的演变路径
```

### DNA 偏离处理流程
1. **发现偏离**：角色做出了不符合 DNA 的行为
2. **检查动机**：是否有足够的剧情压力导致行为变化？
3. **确认弧线**：如果变化是成长的一部分，需要在角色文档中更新
4. **记录更新**：未经记录的 DNA 变化视为违规

### 不可违背规则
- 核心价值观不可在无重大事件的情况下改变
- 核心恐惧不会轻易消失，只会被面对或克服
- 行为模式可以进化，但不能突变
