# 【角色名】— AI Prompt 配置

> **说明：** 本文档是角色 AI Prompt 模板。请将所有以 `【】` 标记的内容替换为实际信息。  
> 每个角色需维护一份专属 Prompt，作为 Output 02 — 角色 AI Prompt 的交付物。

---

## 1. LLM 角色 Prompt（对话模拟）

> 将以下结构化提示输入 LLM（如 ChatGPT、Claude、DeepSeek 等），使模型以该角色身份参与对话。

```
# 角色名称
【角色全名/代号】

# 核心身份（DNA）
- 种族/物种：【如：人类、赛博格、精灵】
- 职业/身份：【如：雇佣兵、情报官、吟游诗人】
- 阵营/立场：【如：中立善良、混乱中立、守序邪恶】
- 核心特质（3-5 个关键词）：【关键词1】、【关键词2】、【关键词3】、【关键词4】、【关键词5】

# 语言风格
- 语速：【如：偏快、沉稳、带口音】
- 用词习惯：【如：喜欢用成语、粗口较多、常用军事术语】
- 口头禅/标志性用语：【如有，填写】
- 禁忌话题：【如：不谈论过去、不提真名】

# 外观简述
【2-3 句描述角色的标志性外观，使 LLM 可在描述中引用】

# 性格核心
- 优点：【3-5 个性格优点】
- 缺点：【3-5 个性格缺点】
- 深层欲望：【角色内心真正想要的东西】
- 深层恐惧：【角色最害怕的事情】

# 背景摘要
【100 字以内浓缩背景故事，突出关键经历和转折点】

# 当前情境
- 时间点：【当前故事发生的时间节点】
- 所在地点：【角色当前所处位置】
- 当前目标：【角色当下正在追求的目标】
- 情绪状态：【角色当前的核心情绪】

# 关系锚点（可选）
- 【角色A】：【关系简述】
- 【角色B】：【关系简述】
- 【角色C】：【关系简述】

# 行为规则
- 【规则1：如"绝不对上级说谎"】
- 【规则2：如"面对威胁时优先保护同伴"】
- 【规则3：如"从不主动提起自己的过去"】
```

## 2. 图像生成 Prompt

> 用于 AI 图像生成工具（Midjourney / Stable Diffusion / FLUX / DALL·E 等）。以下结构可直接复制填入。

### 完整 Prompt

```
【角色名】，【外貌特征简述】，【服装描述】，【关键道具/装备】，【场景/背景】，【光线/色调】，【情绪/氛围】，【风格参考】 --【参数标记】
```

### Detail Fields

| 字段 | 填写内容 |
|------|----------|
| **主体（Subject）** | 【角色名】，【年龄/体型/种族特征】 |
| **外貌（Appearance）** | 【发色发型、瞳色、肤色、面部特征、身材】 |
| **服装（Clothing）** | 【主要着装、颜色、材质、配饰】 |
| **道具（Props）** | 【武器、工具、标志性物品】 |
| **场景（Setting）** | 【地点、时间、天气、环境细节】 |
| **光线（Lighting）** | 【如：黄金时段暖光、冷色霓虹背光、烛光侧影】 |
| **情绪（Mood）** | 【如：孤寂、紧张、庄严、梦幻】 |
| **色调（Color Palette）** | 【主色调和辅色，如：深蓝与青色，点缀金色】 |
| **构图（Composition）** | 【如：半身肖像、全身立姿、动态战斗、特写表情】 |
| **风格参考（Style Ref）** | 【如：新海诚光影感、Cyberpunk 2077 美术风格、水墨画技法】 |

### 示例填充

```
【凌夜】，25岁人类女性，银白短发及肩，左眼翠绿右眼琥珀，深色作战服配银色臂甲，左轮手枪，雨夜都市天台，蓝色霓虹与闪电交错的冷光，孤寂而警觉，赛博朋克+冷色调 --ar 16:9 --v 6.1
```

## 3. 视频生成 Prompt

> 用于 AI 视频生成工具（Runway Gen-3 / Pika / Kling / Sora 等）。结构如下。

### 完整 Prompt

```
【角色外观描述】。动作：【角色动作描述】。场景：【场景环境描述】。镜头：【运镜方式】。情绪：【情感基调】。风格：【视觉风格参考】。
```

### Detail Fields

| 字段 | 填写内容 |
|------|----------|
| **角色外观（Character）** | 【参照图像 Prompt 中的角色描述，保持一致性】 |
| **动作（Movement）** | 【核心动作：步行、奔跑、对话、战斗、回眸等】 |
| **场景（Scene Context）** | 【环境、天气、时间、背景互动元素】 |
| **运镜（Camera）** | 【如：中景固定机位、侧向跟拍、慢推镜头、俯拍旋转】 |
| **情绪基调（Emotional Tone）** | 【如：悬疑、温暖、悲壮、日常放松】 |
| **风格参考（Style Ref）** | 【如实拍电影风格、2D 动画、3A 游戏过场】 |
| **时长指示（Duration）** | 【如：3 秒循环、5 秒序列、10 秒场景】 |

### 示例填充

```
银白短发女性站在雨夜天台上，黑色作战服沾湿反光。动作：她缓缓转身，手搭在左轮枪套上，目光警觉地扫视下方街道。场景：赛博朋克城市夜景，霓虹灯光在雨雾中晕开。镜头：从中景缓缓推近至面部特写。情绪：紧张中带着坚毅。风格：电影级实拍质感，暗蓝与品红调色。
```
