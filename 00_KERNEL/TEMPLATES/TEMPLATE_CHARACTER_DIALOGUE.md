## 二、对白样本表

| 情境 | 原句 | 语气说明 | 用意 |
|------|------|---------|------|
| **日常问候** | 【「早安，今日天气不错。」】 | 【语气平淡，不带感情色彩】 | 【维持基本社交礼仪】 |
| **表达关心** | 【「你脸色很差……要不要休息一下？」】 | 【语调放软，略带犹豫】 | 【展现细腻一面】 |
| **表达愤怒** | 【「你给我闭嘴！这里没有你说话的份。」】 | 【音量骤升，句尾斩钉截铁】 | 【宣示主导权，压制对方】 |
| **表达爱意** | 【「……」→【「笨蛋，我也是。」】（小声）】 | 【先沉默再小声嘟囔，回避眼神】 | 【害羞不擅直白表达】 |
| **表达悲伤** | 【「……没关系，我早就习惯了。」】 | 【声音越来越小，故作坚强】 | 【压抑情绪，不想示弱】 |
| **表达喜悦** | 【「真的？！太棒了——！」】 | 【尾音上扬，语速变快】 | 【毫不掩饰的纯粹开心】 |
| **拒绝** | 【「抱歉，这件事我没办法帮你。」】 | 【礼貌但坚定，不留余地】 | 【划清边界】 |
| **妥协** | 【「……好吧，就这一次。下不为例。」】 | 【叹气后勉强答应，语带无奈】 | 【给对方台阶下】 |
| **开玩笑** | 【「你认真的？哈哈哈，这笑话我记下了。」】 | 【语速轻快，带笑声】 | 【活跃气氛，拉近距离】 |
| **说谎时** | 【「我当然没去过啊。你听谁说的？」】 | 【故作轻松，反问转移焦点】 | 【掩饰心虚】 |
| **压力下** | 【「冷静……深呼吸。我可以处理。」】 | 【自言自语式，短促、节奏化】 | 【自我暗示，稳住情绪】 |
| **最常说的一句话** | 【「唉，随你便吧。」】 | 【百无聊赖/无奈语气】 | 【口头禅，表现随性态度】 |

> 💡 **使用说明**：以上为参考样式，请根据【角色姓名】的实际性格、背景与故事设定替换各栏内容。
> 每个情境可填写多组样本（另起行或扩充为子表格），样本越丰富，AI输出越准确。

## 三、语言风格笔记

### 3.1 标志性口头禅 / 惯用语

- 【例：「不是吧……」——表示惊讶或无奈】
- 【例：「你真是……」——欲言又止，用于吐槽】
- 【例：「讲真」——开启一段认真的发言】

### 3.2 语气词与填充词

- 【例：频繁使用"嗯"、"啊"、"那个……"作为思考缓冲】
- 【例：从不使用语气词，说话干净利落】
- 【例：紧张时会无意识重复"就是说、就是说"】

### 3.3 声线特征（文字可传达的）

- 【例：习惯在句尾上扬，听起来像在质疑】
- 【例：说话带鼻音，给人懒洋洋的感觉】
- 【例：重要词语会刻意放慢加重】

### 3.4 禁忌用词 / 避讳话题

- 【例：从不说粗口，即使盛怒也只说"岂有此理"】
- 【例：避谈【某事件/某人】，提及时会转移话题或沉默】

## 四、对白演变轨迹

| 阶段 | 语言变化 | 原因说明 |
|------|---------|---------|
| **故事初期（S1 / 前段）** | 【语气拘谨，用词正式，句尾多为敬体】 | 【初入陌生环境，保持社交距离】 |
| **故事中期（S2 / 中段）** | 【开始出现口语化表达，语气放松】 | 【建立信任关系，卸下心防】 |
| **故事后期（S3 / 高潮）** | 【语言激烈程度显著增加，情绪波动大】 | 【冲突升级，情感投入加深】 |
| **故事终局（结局）** | 【回归平静，语气沉稳，用词简洁有力】 | 【经历成长，达成内心和解】 |

> 若角色存在"语言能力成长"（如：幼年→成年、异乡人→融入本地），请另附对比例表。

## 五、特殊情境语言规则

| 情境类型 | 说话规则 |
|---------|---------|
| **战斗 / 危急时** | 【句长缩短至5字以内，命令式语气，少主语】 |
| **醉酒 / 意识模糊** | 【句子不完整，逻辑跳跃，重复同一句话】 |
| **对特定人物** | 【对象为【角色A】时会结巴；对象为【角色B】时语气变冷】 |
| **公开场合** | 【使用敬语，措辞谨慎，不流露真实情绪】 |
| **独白 / 内心OS** | 【视角转为第一人称，语气较对白更情绪化、更直白】 |
