# BSOS Visual Language Standard v1.0

> Visual First. Dialogue Second.

---

## Story Rule

所有 Episode 在 QA 阶段必须增加一项检查：

- ✅ **Visual First** — 角色的核心信息是否优先通过视觉语言传达？

---

## 格式

每个角色的视觉语言文件位于 `02_CHARACTERS/CH-NNN_NAME/Visual_Language.md`

```yaml
Visual Symbol:     # 最具辨识度的视觉标记
Signature Action:   # 反复出现的行为动作
Signature Object:   # 与角色深度绑定的物品
Visual Motif:       # 贯穿全剧的视觉叙事语言
Production Rules:   # 视觉语言在剧中的使用规则
```

## Episode Visual Language Checklist

```markdown
- ✅ Visual First（核心信息通过画面传达？）
- ✅ Visual Symbol（角色视觉符号出现？）
- ✅ Signature Action（角色的标志性动作出现？）
- ✅ Production Rules Followed（符合该角色的制作规则？）
```

所有未来的 Episode 必须通过 Visual Language 检查。
